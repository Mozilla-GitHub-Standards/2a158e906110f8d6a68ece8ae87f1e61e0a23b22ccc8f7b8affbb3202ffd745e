Marketing Firebird Website Advertising

Website Advertising

Depending on how aggressive you want to be, there are four basic ways of advertising Firebird on your website. Click on the campaign you wish to pursue for further directions:

Most Aggressive          Roadblock          Popup ad          Buttons          Text Links          Least Aggressive


Roadblock

A roadblock is a page that appears when users of other browsers try to access your site. The page promotes Firebird but allows the user to continue on to your site using their current browser; hence, it acts as a "roadblock" or obstacle. Setting up a roadblock is easy. Simply use Javascript to check the user's browser, and redirect them to the promotional page if they aren't using Firebird: 

if (navigator.userAgent.indexOf("Firebird") == -1) {
  window.location.href="http://www.yoursiteaddress.com/roadblock.html";
}

Popup ad

Opening a popup ad that calls attention to itself and promotes Firebird as a way to block such ads in the future is a delicious and ironic twist. To add a popup to your site for non-Firebird users, use the following Javascript: 

if (navigator.userAgent.indexOf("Firebird") == -1) {
  window.open("http://www.yoursiteaddress.com/popup.html", 'Popup',
              'width=502,height=325,resizable=no,status=no,menubar=no,toolbar=no');
}

Buttons & Banners

The stalwarts of online advertising, these have been around since the days of "Netscape NOW" and will probably make up the bulk of our guerilla marketing campaign. Adding a button or banner to your site requires basic HTML: 

<a href="http://www.mozilla.org/products/firebird/">
  <img src="http://www.yoursiteaddress.com/firebird.gif"
       alt="Download the Mozilla Firebird browser">
</a>
Some of you may want to show buttons or banners only to users of other browsers, which requires the following Javascript: 
if (navigator.userAgent.indexOf("Firebird") == -1) {
  var img = document.getElementById('firebird-ad');
  img.style.display = 'block';
}
and some tweaks to the HTML code above: 
<a href="http://www.mozilla.org/products/firebird/">
  <img id="firebird-ad" style="display: none;"
       src="http://www.yoursiteaddress.com/firebird.gif"
       alt="Download the Mozilla Firebird browser">
</a>
You are welcome to use the buttons and banners below on your site. If you choose to design your own, please submit it so we can offer it to other people!


Text Links

To add a link promoting Firebird, insert the following simple HTML: 

  <a href="http://www.mozilla.org/products/firebird/">Download the Mozilla Firebird browser</a>
Feel free to change the wording however you see fit (for example, you might just want it to say "Free popup-blocking software").
