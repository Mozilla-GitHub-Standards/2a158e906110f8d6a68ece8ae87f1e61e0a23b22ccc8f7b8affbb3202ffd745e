  • Foundation Employees and staff@mozilla.org  (adminstrative arm)
    • Creates policies & procedures, function as a switchboard, facilitate cooperation, deligate!
  • drivers@mozilla.org  (technical arm - focus and guide)
    • Build development (Milestone) plans, execute Milestone checkpoints
  • Super-Reviewers (Strong engineers with broad experiance in the code)
    • Review changes for effects on architecture, use of interfaces, I18N, security,  performance, footprint, code stds, & overall quality
  • Mozilla Module Owners (Recognized experts for ~90 subsytems)
    • Network, Layout, to UI and Build Config ...everything to build a browser
  • Code Contributors
    • ~400 engineers have access to CVS -- about 75 active in any given month
    • Thousands have submited suggested patches as attachments to bugs
  • QA & Testing Contributors
    • 10-20k testers download daily builds and actively test
    • watch engineers and bugs -- report between 75 and 100 bugs each day
  • Extension and Plugin Developers - All major plugins supported.  Over 145 extensions
  • Localization projects -- Mozilla translated to 99 languages
  • Porting teams -- Mozilla runs on more than 12 OS/Hardware platforms
  • Distributors and Enterprise Deployments -- work with staff, drivers, and contributors
