Not a Primer
Many good backgrounders available
Most simple use cases use XML RDF datasources (.rdf)

nsIRDFDataSource
API providing graph traversal/manipulation
Various ways to implement this API
Most common is to just use .rdf – but not suitable for large data sets
Therefore we can use a custom back
end.

Some Examples…
Bookmarks – a RDF datasource whose underlying format is a .html file
History – a RDF datasource whose underlying format is a Mork Database.
Anything is possible – just implement nsIRDFDataSource as the front end.
