status update

maintained by Alex Bishop <alex@mozillazine.org>

Last Updated Friday March 28, 2003

This status update page is updated approximately once a week. To get news in between these updates, you are invited to check out MozillaZine and MozillaNews, two sites devoted to Mozilla news and advocacy.

Previous Updates

Minotaur

Scott MacGregor has formally announced the Minotaur project to the Mozilla community.

ChatZilla

ChatZilla 0.8.24 (released earlier this month) has now landed on the Mozilla trunk.

JavaScript Debugger (Venkman)

Venkman 0.9.53 has landed on the trunk. Amongst other improvements, this version allows you to change values from the Local Variables or Watch views, add watches from the Local Variables view and save and restore breakpoints and watches.

Calendar

A new Windows build of Calendar that works with 1.4 trunk nightlies is available.

Smooth Scrolling

Mozilla now supports optional smooth scrolling, controlled by the boolean general.smoothScroll pref (bug 174049).

Bookmarks Branch

The bookmarks branch has now landed on the trunk. This rewrite has fixed many longstanding issues (bug 196756).

Cookie Code Rewrite

The second stage of the cookie code rewrite has now landed (bug 195908).

Permission Code Rewrite

Mozilla's permission code (used for managing cookie, image and popup permissions) has been rewritten (bug 191380).

NTLM Authentication

Mozilla now supports NTLM authentication on Windows using the SSPI API (bug 159015).

Popup Blocking

Mozilla's popup blocking has been simplified to support only whitelisting only (similar to Phoenix) (bug 195924).

Junk Mail Controls

There is now an option in the Tools menu of Mail & Newsgroups that allows you to delete all junk mail in a folder (bug 194918). You can also now manually run Junk Mail Controls on a folder (bug 198749).

Personal Toolbar Overflow Handling

The Personal Toolbar now displays a chevron if there is not enough space to show all the available bookmarks. Clicking this chevron displays a menu of the excess items (bug 66919).

Modern Theme Toolbars

The Modern theme now supports displaying toolbars as pictures and text, pictures only and text only (bug 171013). Note that in Navigator the first two options are visually indistinguishable as Modern's Navigator toolbar icons do not have text below them.

Tree Status

The tree is now frozen for Mozilla 1.4 Alpha.

Stuff That Should Have Been in the Last Update

  • NSS 3.8 has landed on the trunk (bug 197874).
  • P3P is now built by default (bug 198089).

Staff Meeting Minutes

Minutes from the mozilla.org staff meetings of Wednesday 19th March are now available.

Independent Status Reports

Third party updates from Friday 28th March are now available.
