This status update contains news on Mozilla 1.7 Alpha, Minimo 0.1, the string code, layout, large downloads, support for Sun keyboards and more.

This status update contains news on Mozilla Firefox (formerly Firebird), Mozilla Thunderbird, junk mail detection, browser data migration, popup blocking, SVG, new mail notification, permissions and more.

This status update contains news on Mozilla 1.6, the Mozilla Development Roadmap, Camino, relicensing, the URL spoofing vulnerability, CSS and more.

This status update contains news on Mozilla Firebird, ChatZilla, history searches, proxy configuration, internationalized domain names and more.
