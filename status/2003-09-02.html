status update

maintained by Alex Bishop <alex@mozillazine.org>

Tuesday September 2, 2003 (Status Update #212)

This status update page is updated approximately once a week. To get news in between these updates, you are invited to check out MozillaZine, a site devoted to Mozilla news and advocacy.

Previous Updates

Mozilla 1.5 Beta

Mozilla 1.5 Beta was released on Wednesday 27th August. Amongst the usual improvements, there's a range of Mail & Newsgroups enhancements and a major ChatZilla update. This is also the first milestone to include a spellchecker.

Mozilla Thunderbird

Several rounds of Mozilla Thunderbird 0.2 release candidate builds have been made available. The latest candidates can be found in ftp.mozilla.org/pub/thunderbird/nightly/2003-09-01-trunk, with earlier test builds in ftp.mozilla.org/pub/thunderbird/nightly/2003-08-26-trunk. Scott MacGregor has posted information about the first set of candidates, second round of builds and most recent test binaries to the MozillaZine Forums. The Mozilla Thunderbird Roadmap has more information about future plans.

Composer

Daniel Glazman has checked in the first files for the standalone version of Composer.

ChatZilla

The CHATZILLA_COMMANDS_YOU branch has landed, updating the version of ChatZilla distributed with the Mozilla Application Suite to 0.9.28 and fixing no less than 33 bugs (bug 210744). The Mozilla IRC client now makes use of parts of the Venkman Application Framework.

JavaScript Debugger (Venkman)

Venkman 0.9.78 has landed on the trunk, replacing the 0.9.75 version (bug 216452). Amongst other improvements, this version makes it easier to set and clear breakpoints and fixes the problem where the cursor would be left one character before the end of the line after using the up arrow to recall a previous command.

HTML Message Composition

Sending mail or newsgroup messages with background images now works again (bug 116867 and bug 170504).

MozillaZine's Fifth Anniversary

Leading Mozilla news and advocacy site MozillaZine, launched on Tuesday 1st September 1998, celebrated its fifth birthday this week. The site has a special section to mark the anniversary.

Mozilla Links Newsletter Launching Soon

The new Mozilla Links Newsletter is set to launch on Tuesday. The bi-weekly publication, which has spawned from the nascent Mozilla marketing effort, will feature headlines, tips and information about third-party projects. Subscribe now by sending a blank email to newsletter-request@mozilla.org with 'subscribe' in the subject line. Issue one will be sent out on Tuesday 2nd September.

Status Updates

This is the first status update for quite some time. Apologies and thanks to everyone who emailed. This update aims to cover most of the major developments that have occurred since the last one came out.

Tree Status

The trunk is currently frozen for Mozilla 1.5. Check tinderbox for the latest status.

Stuff That Should Have Been in the Last Update

Linux builds of Mozilla Firebird compiled with GTK 2 and XFT are now available from ftp.mozilla.org/pub/firebird/nightly/latest-trunk.

Staff Meeting Minutes

Minutes from the mozilla.org staff meeting of Monday 28th July, Monday 18th August and Monday 25th August are now available.

Independent Status Reports

Third party updates from Monday 11th August and Friday 22nd August are now available.
