status update

maintained by Alex Bishop <alex@mozillazine.org>

Last Updated Friday May 2, 2003

This status update page is updated approximately once a week. To get news in between these updates, you are invited to check out MozillaZine, a site devoted to Mozilla news and advocacy.

Previous Updates

Mozilla Firebird

Mozilla Firebird now includes Quality Feedback Agent (Talkback), allowing crash data to be sent back to the developers for analysis.

JavaScript Debugger (Venkman)

Venkman 0.9.67 has landed on the trunk (bug 203848). As well as bug fixes, this version includes a new tabbed views feature that allows views to be 'stacked'. Newer versions are available from the Venkman Development page.

Profiles

Backend changes have been made to allow profiles to be shared (bug 202474).

Message Views

Several issues with the Mail & Newsgroups message views feature have been fixed (bug 189543, bug 187990 and bug 190187).

Navigator Startup, New Window and New Tab Behavior

The Preferences UI for controlling Navigator startup, new window and new tab behavior has been simplified (bug 201177).

Automatic Image Resizing

When hovering above an image that can be resized using automatic image resizing, a custom magnifying glass pointer is used (not currently supported on Mac OS X) (bug 189719).

Bookmarks

Importing and exporting bookmarks now works again (bug 200359).

GTK 2 nsITheme Support

nsITheme support is now built by default in GTK 2 Mozilla builds (bug 174471).

Windows Integration

On Windows, setting Mozilla to be the default application for opening supported file types (for example, HTML documents or GIF images) now changes the icons of those file types to Mozilla icons (bug 44714).

Tree Status

The trunk is still frozen for Mozilla 1.4 Beta. Check tinderbox for the latest status.

Stuff That Should Have Been in the Last Update

Bugzilla 2.16.3 and 2.17.4 have been released. The former release is the latest stable version while the latter is a development snapshot. Both fix security flaws; see the security advisory for more details. A new Bugzilla Status Update is also available.

Staff Meeting Minutes

Minutes from the mozilla.org staff meetings of Monday 14th April and Monday 28th April are now available.

Independent Status Reports

Third party updates from Monday 28th April are now available.
