Kin

  • Lots of code reviews.
  • Helped investigate Linux tree blocker issues.
  • Checked in fix for bugs 82556, 82680, and 81078.
  • Finally figured out what was causing bug 74383 (textarea input form crashed during complex edit session) . This is the bug that is causing text to disappear and random garbage to render in textareas and textfields. It seems this is due to the fact that nsBoxToBlockAdaptor::Reflow(), in certain situations, suppresses reflows that are needed to update text frames that have changed, and create new continuation frames. See bug 74383 for more details. It is currently assigned to evaughan@netscape.com.
  • Helped mjudge debug 80048 (First letter not selectable in textbox if particular letter) . I couldn't repro it on my machine, but we could at least compare a debug NT build with his opt Win98 build. Turned out to be a regression introduced by some BIDI code in the font metrics code.
  • Triaged and did some preliminary debugging of various editor bugs.
