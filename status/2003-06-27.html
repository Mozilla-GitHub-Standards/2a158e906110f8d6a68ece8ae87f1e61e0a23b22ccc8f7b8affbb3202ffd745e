status update

maintained by Alex Bishop <alex@mozillazine.org>

Friday June 27, 2003 (Status Update #205)

This status update page is updated approximately once a week. To get news in between these updates, you are invited to check out MozillaZine, a site devoted to Mozilla news and advocacy.

Previous Updates

Mozilla 1.4 Release Candidate 3

The third release candidate of Mozilla 1.4 is now available. The final version of Mozilla 1.4 is expected on Monday.

Mozilla 1.5 Alpha

Asa Dotzler has announced that an extra two weeks has been added to the Mozilla 1.5 Alpha milestone period to account for the current development focus on 1.4. The milestone schedule section of the Mozilla Development Roadmap has been updated to reflect this change, with the new freeze date set to Wednesday 9th July.

Composer

Many of the Composer++ changes have now landed on the Mozilla trunk. Composer now allows blocks be absolutely positioned (with support for resizing and z-index modification), has a snap-to-grid feature for positioned elements and there's a new UI for inline table editing (also enabled in HTML mail composition). See Daniel Glazman's weblog posting for more details.

Mozilla Thunderbird

The mail composition window of Mozilla Thunderbird now has a Contacts Sidebar panel, which allows users to easily select the addresses to send their emails to. The feature is available in the new 2003-06-25 nightly of Thunderbird. More information about this build can be found in Scott MacGregor's announcement to the MozillaZine Forums.

ChatZilla

ChatZilla 0.8.34 has landed on the trunk. See the ChatZilla page on hacksrus.com for more information, including a list of bugs fixed and installation links.

Tabbed Browsing

A window with more than one tab open will now display a confirmation dialog before closing (bug 108973).

Mail/Newsgroup Message Date Header Display

Setting the boolean mailnews.display.original_date preference to true makes Mail & Newsgroups display message dates in their original format (complete with time zone offset) (bug 118899).

DOM Inspector

The bitmap/PNG, screen capture and box model color-picking functionality has been removed from the DOM Inspector because it was broken (bug 210096). This functionality will be restored if it can be made to work correctly again (Uzilla, LLC has contracted the Mozdev Group, Inc. to work on this).

Tree Status

The trunk is currently open (but check tinderbox for the latest status). Checkins to the 1.4 branch require approval from drivers@mozilla.org.

Stuff That Should Have Been in the Last Update

The third stage of the cookie code rewrite has landed (bug 200632).
