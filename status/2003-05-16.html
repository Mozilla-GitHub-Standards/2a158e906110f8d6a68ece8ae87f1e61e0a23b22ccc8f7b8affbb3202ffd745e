status update

maintained by Alex Bishop <alex@mozillazine.org>

Friday May 16, 2003 (Status Update #199)

This status update page is updated approximately once a week. To get news in between these updates, you are invited to check out MozillaZine, a site devoted to Mozilla news and advocacy.

Previous Updates

Mozilla Firebird 0.6

Mozilla Firebird 0.6 has been released. Changes include a new default theme based on Qute by Arvid Axelsson, a new Options dialogue, context menus for Bookmarks menu items, automatic image resizing and smooth scrolling. This release also includes Quality Feedback Agent (Talkback), which allows crash data to be sent back to the developers for analysis.

Camino

Camino now has a blocked popup indicator similar to those found in many other Mozilla-based browsers.

Composer++

Daniel Glazman has released some special builds of Mozilla Composer with extra functionality. These new features will land on the trunk soon.

Documentation

  • A new How to Document Mozilla document has been published.
  • The information about contributing documentation has also been revamped.

XPCOM

Alec Flett has started working on some XPCOM documentation.

nsHashtable Officially Deprecated

The old nsHashtable has been officially deprecated in favor of the more heap-friendly nsTHashtable.

Xprint

Roland Mainz writes in with news about the release of Xprint version 008: "Just update/install your Xprint server and get TrueType font and CUPS/LPRng support for free — works for all Mozilla versions (incl. Netscape 7.x, Phoenix/Firebird, Thunderbird and DocZilla) since 0.9.5 without the need for an update on the Zilla-side."

JavaScript Debugger (Venkman)

The latest Venkman version, 0.9.75, has landed on the trunk (bug 204572).

Junk Mail Controls

Several miscellaneous improvements have been made to Mozilla's junk mail controls (bug 180153, bug 191042, bug 194273, bug 204679 and bug 205054).

Focus Problems

Several focus problems have been fixed (bug 201996, bug 200735, bug 203117 and bug 194104).

Digest Authentication with SSL Proxies

Digest authentication now works with SSL proxies (bug 203813).

Windows UNC-Based Profiles

UNC-based profiles should now work better on Windows (bug 162025). Note that there are still several problems with UNC file access on some versions of Windows (bug 101953).

Tree Status

The trunk is still frozen for Mozilla 1.4. Check tinderbox for the latest status.

Staff Meeting Minutes

Minutes from the mozilla.org staff meeting of Monday 12th May are now available.

Independent Status Reports

Third party updates from Thursday 15th May are now available.
