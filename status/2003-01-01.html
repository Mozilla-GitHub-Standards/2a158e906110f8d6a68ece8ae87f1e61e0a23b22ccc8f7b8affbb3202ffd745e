maintained by Tim Rowley <tor@cs.brown.edu>

Last Updated Wednesday January 1, 2003

This status update page is updated every week. To get updates and news throughout the week you are invited to check out mozillaZine, a site devoted to Mozilla advocacy.

mozilla.org timeline

It's been a busy year for mozilla.org, with 35 seperate releases. Below are links to their release notes and some selected highlights of the changes.

  • January 1, 2002 - First bug filed in the year: 117591
  • February 4, 2002 - Mozilla 0.9.8
    • nsITheme giving native classic widgets on Mac OS X and WindowsXP
    • MNG support turned on
    • Dynamic themes turned on
    • favicon.ico turned off (still supported with <link>)
  • February 13, 2002 - Chimera 0.1
  • February 14, 2002 - Chimera 0.11
    • 33% faster than 0.1
  • February 22, 2002 - Chimera 0.12
    • Scroll wheel support
    • Key events
    • Hover and cursor feedback
    • Improved scrolling speed
  • February 24, 2002 - Chimera 0.1.3
    • Security (SSL, HTTPS) support
    • Alert and user/password prompts
    • 5% faster than 0.12

Third party gecko integration timeline

In 2002 we started seeing the Mozilla layout engine, gecko, being embedded in various commercial projects. Here's some that hit the news:

  • January 2, 2002 - OEone HomeBase 1.0
    • Integrated Linux, XML based webtop and applications
    • Additional released throughout the year (see OEone's news page)
    • Version capable of running on a Red Hat installation released in August.
  • March 14, 2002 - AOL with Gecko Beta
  • April 16, 2002 - Compuserve 7
    • Integrates Gecko instead of Internet Explorer
  • May 15, 2002 - Netscape 6.2.3
    • Based on Mozilla 0.9.4
    • Fix for XMLHttpRequest security problem
  • May 16, 2002 - AOL for Mac OS X testing
  • May 22, 2002 - Netscape 7.0 Preview Release 1
    • Based on Mozilla 1.0rc2
  • June 10, 2002 - Beonex Communicator 0.8
    • Based on Mozilla 1.0
    • Mozilla tweaked for privacy and security
  • August 12, 2002 - AOL for Mac OS X
    • Integrates Gecko instead of Internet Explorer
  • August 29, 2002 - Netscape 7.0
    • Based on Mozilla 1.0.1
    • First Netscape release based on post-1.0 code
    • Above the standard 1.0.1 features, it includes access to Netscape Webmail and AOL accounts, P3P support, AIM/ICQ support, Radeio@Netscape, and a new throbber
  • September 25, 2002 - ActiveState Komodo 2.0
    • Mozilla-based IDE for editing, debugging, and testing applications written in Perl and Python
  • September 30, 2002 - IBM Web Browser for OS/2 V2.0
    • Based on Mozilla 1.0.1
  • November 1, 2002 - K-Meleon 0.7
    • Gecko based browser for Windows
    • Based on Mozilla 1.2beta
    • Rewritten bookmark handling, improved macros, layers for tabbed browsing
  • December 10, 2002 - Netscape 7.01
    • Based on Mozilla 1.0.2
    • New pop-up blocker
    • Start page can now be a bookmark group

Additionally, most Linux distributions include some version of Mozilla.
