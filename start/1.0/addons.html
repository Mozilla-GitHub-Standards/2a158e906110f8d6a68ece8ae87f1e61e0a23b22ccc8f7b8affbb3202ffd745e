Localizations You can change the Mozilla user interface from English to any of these languages.
Currently available for Mozilla 1.0: Asturian, Basque, Breton, Simplified Chinese (China), Traditional Chinese (Taiwan), Danish, Dutch, English (United Kingdom), Esperanto, Estonian, French, Galician, German, Greek, Hebrew, Hungarian, Italian, Japanese, Korean, Malay, Norwegian Nynorsk, Polish, Portuguese (Brazil), Romanian, Russian, Scots, Slovak, Sorbian, Spanish (Argentina), Spanish (for Latin America), Spanish (Spain), Turkish, Ukrainian and Welsh.
Currently available for Mozilla 1.0.1: Simplified Chinese (China), Traditional Chinese (Taiwan), Czech, Danish, English (United Kingdom), Estonian, French, German, Greek, Hebrew, Italian, Japanese, Korean, Malay, Polish, Romanian, Slovak, Slovenian, Sorbian, Spanish (Argentina), Spanish (Spain) and Ukrainian.
Currently available for Mozilla 1.0.2: Simplified Chinese (China), Czech, Esperanto, English (United Kingdom), German, Greek, Italian, Japanese, Polish, Romanian, Slovak, Sorbian, Spanish (Argentina), Spanish (Spain) and Ukrainian.

The number of available languages continuosly increase, so check back later if yours is not listed yet.

See the Mozilla Localization Project pages for more details and information on how to contribute a localization in your language, and the complete list of contributed localized material.
