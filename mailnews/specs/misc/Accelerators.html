Notes

  • Items in orange are accelerators that have different functions in different components. Not recommended if avoidable, but acceptable if history or necessity dictate.
  • Items in red are accelerators that have different functions in different components and are CONFLICTS.
  • Items in italics are not yet implemented.

Alt - Alt on Windows and Linux, Option on MacOS (unless stated otherwise). 
Ctrl - Ctrl on Windows and Linux, Cmd on MacOS (unless stated otherwise).
