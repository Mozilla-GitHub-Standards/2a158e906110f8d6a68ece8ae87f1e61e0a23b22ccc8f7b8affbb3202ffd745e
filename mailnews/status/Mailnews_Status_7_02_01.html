Mailnews Status 7/2



Mailnews Accomplishments:
 
  • Resolved as fixed 31 bugs.
  • Jean-Francois has a fix for a bug where some messages with file urls in them can't be sent.
  • Cavin has a fix for making it so that local folders with names that contain characters that are illegal for that platform's file system now load correctly.  He also found a number of rename bugs while working on this.
  • Scott redesigned the helper app UI and landed it.  He updated and landed his changes for showing SMTP progress in the compose progress window on the trunk.  He finished a bin hex stream decoder for windows and linux, made the collapsed header view persistent, and added a service for deleting temporary files on exit (particularly those generated by helper apps).
  • Navin made it so that the New Folder dialog for servers that support folder/message only folders shows up better than it did before.  He fixed a problem copying messages from pop to imap.  He worked on improving the thread pane drag and drop code and has a fix for undo not working in the standalone window.
  • Candice has a fix for making it so that when we import Outlook address books we get the email addresses.
  • David has fixes for imap empty trash not working offline, for moving an offline message while offline and copying the offline body to the desination folder, and for imap saving as template not working offline. He checked in the fix for a case where imap messages could disappear from a folder.  He also fixed corrupt/invalid db handling for imap and news folders.
  • Varada checked in a few UI changes into the trunk that we'll try to land on the branch. He worked with Jean-Francois on the case where some messages with file urls in the body aren't sending. 
  • Bhuvan removed the duplicate diskpace pref from the pop server panel. 
  • Seth added newsgroup search on the trunk.  He made it so news searches don't fail if there's an '@' .  He also made it so news is threaded by default.
  • Gervase Markham made it so that the copy email address context menu shows up in plaintext messages.
  • Bradley Hart fixed some wording in the Account Setup dialog.
  • Håkan Waara improved some wording in the filters dialog.
  • Blake Ross turned search into a window and removed the close button.
