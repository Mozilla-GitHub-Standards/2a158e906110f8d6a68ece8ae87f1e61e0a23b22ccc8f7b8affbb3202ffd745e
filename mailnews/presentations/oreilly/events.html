Mail Events

  • IMAP, NNTP, POP download data into local database
  • Database updates in-memory mail objects: folders, messages
  • Folders broadcast changes about folders and messages
  • Mail Session rebroadcasts to global folder listeners
  • RDF is a global folder listener, updates UI
  • Other event listeners: folder load complete, message copy complete, etc
