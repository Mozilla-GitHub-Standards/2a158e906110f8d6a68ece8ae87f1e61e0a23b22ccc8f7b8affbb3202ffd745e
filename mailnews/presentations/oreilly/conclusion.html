Conclusion

  • Fully scriptable with JavaScript
  • Flexible architecture - pluggable protocols (almost)
  • Completely cross-platform - Mozilla hides all platform issues
  • Does not bloat when not loaded
  • Easily customized - skin, XUL, etc
We can use some help!
  • Lots of polish/cleanup work - easy introduction to XUL, etc
  • Lots of small features - advance to next folder, etc
  • Larger features - Search/Filters, Address book
  • And of course, bugs!
