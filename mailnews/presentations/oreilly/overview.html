Mail/News Components

  • Mail Reader - heart of mail client.
    • Base protocol library - IMAP, POP, NNTP - each has a unique tag string
    • Multiple Accounts, Servers, Folders
    • News and Mail treated identically in most of base client.
    • Search/Filters
  • Mail Compose - for sending mail messages
    • Additional protocol: SMTP
    • Use of Ender (editor) widget
    • Multiple Identities, Attachments, etc
  • Address Book
    • Not really dependant on mail
  • ALL interfaces are scriptable!
