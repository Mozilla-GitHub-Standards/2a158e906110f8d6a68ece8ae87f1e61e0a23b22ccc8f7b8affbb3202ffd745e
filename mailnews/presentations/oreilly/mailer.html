Mozilla Mail

  • Key Services - mostly singleton objects

    • View Navigation - navigation of messages in folders
    • Protocol Information - per-protocol
    • libmime - streaming/conversion
  • Common Objects
    • Accounts, Servers, Folders, Threads, Messages
    • Message Database Indexes - one per folder
    • DataSources - Adapter from Mail to RDF
    • URL objects
    • Mail Windows, Status Feedback
