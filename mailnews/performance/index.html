Mailnews Performance and Footprint Page
 
Performance is a key issue for mailnews.  Some of the areas we are currently focusing on include folder loading, thread pane scrolling, message display, sorting, message deleting, replying to a message and starting up mail windows.  There are other areas that can use tuning as well.  In addition we'll need to start addressing footprint problems.  If you are interested in finding out more about these issues and helping out, read the documents on this page and post to netscape.public.mozilla.mail-news.
