About This Document
Use this first section to describe the document itself. For example, this template is used for creating new documentation as a part of mozilla.org's Open Source Documentation Project. Note that the font size in this document does not vary much at all (particularly within sections), that no external stylesheet is used, and that the sections are clearly divided in the body and referenced in a list at the top. Basic stuff.

Use the headers to separate topics. Technical, sometime-authors often do not distinguish different topics of information within a document from one another clearly. Hopping from one abstruse subject to another, authors do not give the reader enough clues about the topic at hand, about the progression of things. How many hyper-technical documents would benefit from a simple breaking up into chunks! Find out what's in your document and divide it up into these big, blue sections. It looks nice, and it's clear where each section is.
