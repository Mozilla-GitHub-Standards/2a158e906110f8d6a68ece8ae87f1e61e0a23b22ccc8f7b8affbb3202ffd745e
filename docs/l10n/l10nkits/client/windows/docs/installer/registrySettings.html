Windows Registry Settings

This document is provided as a reference to the Windows registry settings implemented by Communicator and Navigator. You may use this as a guide to the registry settings you must implement if you create your own installation program (as opposed to localizing an existing Netscape installer).

Note that in the Netscape installers, these settings are implemented only within .INI files. This simplifies modifying and localizing the installer. It is recommended that your installer do the same.

You may observe the registry changes made by any product with tools such as Regmon. This may be a useful addition to your toolset.

These notes are applicable to Netscape Communicator version 4.03.
