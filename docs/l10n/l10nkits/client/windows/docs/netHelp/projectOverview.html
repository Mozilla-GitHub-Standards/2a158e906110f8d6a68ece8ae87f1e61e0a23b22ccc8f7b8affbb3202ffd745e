1. Project Overview

Welcome to the Netscape NetHelp Localization Kit!

Welcome to the third revision of the first incarnation of the Netscape NetHelp Localisation Kit!
This kit will guide you through the process of localising the NetHelp source files which make up the Communicator 4 help content and the JavaScript engine on top of which NetHelp is implemented.

NetHelp is a completely new way of implementing on-line help in Netscape's client product and most likely the reader of this document is as new to the process as the author is. As such, please take the time to familiarize yourself with the workings of NetHelp by running it from the "Help | Help Contents" menu of Communicator. For additional background, please check out the reference documents mentioned below.
