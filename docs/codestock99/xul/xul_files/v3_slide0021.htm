XUL Benefits
• web application (server-resident) and client
application (locally-installable) developers
– easy to build cross-platform, cross-device user
interface quickly
– XUL adds common UI metaphors (boxes,
springs) that web standards lack
– can build small, powerful, quickly-downloadable
application by leveraging power of browser
instead of duplicating it
– throw away your books on Windows, Mac, and
Motif or GTK UI development: use lightweight
web standards and markup languages instead
