Existing (Nav4) Plug-in
Binaries
• Existing Nav4 plug-in binaries work as-is in
Mozilla-based browsers such as Nav5
• Existing plug-in content still works
• If content makes LiveConnect calls from
JavaScript, those calls will do nothing due to
lack of old Netscape JVM
• If plug-in has LiveConnect API, upgrade to
Mozilla plug-in API to ensure backward
compatibility with existing content’s
LiveConnect calls
