How Can I Format XML
Documents? XML + CSS1
• Use CSS1 to format your XML documents.
• Associating Style Sheets with XML
Documents (W3C Recommendation)
• Put a processing instruction in your XML
file to import the style sheet:
<?xml-stylesheet href="mystyle.css"
type="text/css"?>
• Put CSS rules in the style sheet file:
Airline {
  display: block;
  font-style: italic;
}
