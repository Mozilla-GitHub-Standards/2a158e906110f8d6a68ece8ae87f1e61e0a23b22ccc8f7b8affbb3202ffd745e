A Look at the Mozilla Technology and Architecture
  XPToolkit: Creating user interface
  • Cross-Platform Toolkit (XPToolkit)
  • Widgets and markup/CSS for creating user interface appearance and behaviour
  • Can specify UI appearance and behaviour using XML, CSS, and JavaScript/C++
  • skins/themes change appearance only, using images and CSS
  • Can also create whole new applications (e.g. Aphrodite)
