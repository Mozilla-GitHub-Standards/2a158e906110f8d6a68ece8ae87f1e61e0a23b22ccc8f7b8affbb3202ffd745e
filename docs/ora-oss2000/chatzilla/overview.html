  • Introduction
    • Robert Ginda <rginda@netscape.com>
  • What is Mozilla? (you've probably heard this.)
    • The XPToolkit (XUL) provides the UI widget set.
    • XPCOM provides a COM implementation.
    • XPConnect makes it possible for arbitrary XPCOM components to be controlled from JavaScript.
  • What is ChatZilla?
    • ChatZilla is an IRC client written entirely in JavaScript, using XUL and CSS to describe its UI.
    • Screenshots of the progress of the UI are available at http://www.ndcico.com/jsirc/screenshots/.
