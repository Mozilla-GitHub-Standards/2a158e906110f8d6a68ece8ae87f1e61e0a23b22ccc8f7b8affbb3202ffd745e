  RDF Data Sources [ << Previous | Next >> ]

  • ChatZilla uses the In Memory Data Source to hold user information.
  • RDF Data Sources are objects which contain RDF data, and can be queried for that data.
  • RDF data sources can be read/write, depending on the implementation.
  • Any object which implements the nsIRDFDataSource interface can be treated as an RDF data source.
  • Some data sources of interest are...
    • An in memory data source can be used to build a graph of RDF data at runtime.
    • A remote data source can be used to retrieve data from files and urls.
    • A composite data source can be used to combine data from multiple datasources, giving the appearance of a single coherent source.
    • Also, there are various component-specific implementations scattered throughout the tree. These provide RDF-like views of native data without the overhead of the (generalized) in memory data source.
