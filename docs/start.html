Welcome to the Bleeding Edge

Are you in the Right Place?

If you're looking for a more stable release of Mozilla, this is not the build for you. You should try downloading the latest milestone.

If you're looking for the very latest, but potentially unstable, build of Mozilla, this is the browser for you!

Not sure which build of Mozilla you're looking for? Read about the difference between Milestone builds and Nightly builds.

If you're reading this, you've probably downloaded the latest Mozilla Nightly Build. Thanks for downloading and using this build.

Before you get started with this build

You may want to check out recent posts in the builds newsgroup. It's a good place to find out about any major problems have been found with today's build.

How you can contribute to Mozilla by using this build

  • Report any bugs you find. Use the search form to find out if your bug has already been reported. If you can't find it, head over to Bugzilla, mozilla.org's bug tracking system, and follow the steps under "Before reporting a bug".
  • Test recent changes, and verify bug fixes.
  • Join the BugAThon. If you can script, you can help simplify test cases.
  • Run tests. See mozilla.org's quality pages for test cases. Also check out the netscape.public.mozilla.quality newsgroups, where calls for testing help will be posted.
    • netscape.public.mozilla.qa.general
    • netscape.public.mozilla.qa.browser
    • netscape.public.mozilla.qa.editor
    • netscape.public.mozilla.qa.i18n
