Summary

    This document describes the design for the context menus in Netscape Communicator. We discuss the following:
    • Areas of consistency

Design Goals

  • Keep the menus simple.
  • Menus must be consistent across components.

Target User

    Anyone who uses Communicator.

User Tasks

    Fundamental Tasks
    • Access menu items with the context menu
    • Have quick and easy access to menu items frequently used
    • Find menu commands when needed
