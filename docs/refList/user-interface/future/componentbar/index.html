Summary

    The platforms that Communicator supports provide numerous ways for applications to be accessed quickly. For instance, within Win95/NT the MS office shortcut bar allows for shortcuts for frequently used applications to be placed on the bar. On the Macintosh, the control strip provides a launching point for the Communicator suite. Communicator should utilize these various starting points within the many platforms to provide our users with access to the various components of the Communicator suite. The shortcut area could provide the same function as the component bar within the Communicator package, providing accesibility and notification from within the operating system environment.

Design Goals

  • Provide quick and easy access to various components of Communicator
  • Use existing OS tools to launch Communicator components
  • Enhanced presence within computing environment (os)

Target User

    The target user for this feature is any user who runs one or more of the applications in the Communicator suite.

User Tasks

    Fundamental Tasks

    • Understand the purpose and function of the access to Communicator
    • Access various components

Macintosh

    The control strip on the Mac is typically used to access control panels. However, it should be possible to attach a control strip module that allows access to the various components.

    Another idea for a Mac specific solution would be to create a menu item that is the Communicator icon which would exist on the right hand side of the desktop, next to the finder menu. When expanded, the menu would show the six components of Communicator.

Macintosh Control Strip

Windows

    When Communicator is installed, an icon should appear in the Windows95/NT taskbar next to the Start button. On mouse down, a menu containing all the Communicator components should appear. The user should be able to select one of the components to launch it.
Windows 95/nt taskbar implementation

    Another solution specific to the Windows95/NT platform is to create a new program group on install in the Microsoft Office shortcut bar. Users can change the view to the Communicator set and launch any of the components from here.
