Overview

SpiderMonkey is the code-name for the Mozilla's C implementation of JavaScript, now up to version 1.7. See www.mozilla.org/js and mozilla development center for further information about JavaScript, and JavaScript 1.7 for further information about version 1.7.

JS 1.7.0 Release

This release for JS 1.7.0 is based on the FIREFOX_2_0_0_8_RELEASE CVS tag. It can be downloaded from ftp://ftp.mozilla.org/pub/mozilla.org/js/ or its mirror sites.

You can check out this release directly from CVS using the tag JS_170.
