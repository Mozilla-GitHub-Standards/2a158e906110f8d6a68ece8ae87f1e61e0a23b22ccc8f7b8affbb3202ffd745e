Windows98 WindowsME WindowsNT Windows2000, Linux RedHat 6.2, Solaris

Install JRE 1.3.0_01 if you dont have it already. You can get it from http://java.sun.com/j2se/1.3.0_01/.Click on the appropriate button for your platform. After the XPInstall completes,close the browser.



The Blackwood XPIs also include examples for the various Blackwood components. For information on running these examples, please refer to the README file in the javadev directory which will be created in your Netscape 6 installation directory. Information on PATH and other Environment variable settings is in the Env settings file.


For Pluglets, we have included some sample pluglets in the package. These include a Java Media Player and a SVG (Scalable Vector Graphics) Viewer.  

For viewing the JMFPlayer pluglet examples, you will need to download and install Java Media Framework and add the sound.jar and jmf.jar JAR files to your CLASSPATH.

For viewing the SVGViewer pluglet examples, you will need to download and install the BATIK JAR file. This is a JAR file containing the compiled sources published by the BATIK project at http://xml.apache.org/batik/. This JAR file is approx. 4.5 Mbytes small.
