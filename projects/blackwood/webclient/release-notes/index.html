Releases

These are the links to download packages and notes for Milestone Releases of the Webclient Project. The Webclient Milestone numbering scheme is independent of the Mozilla Milestone numbering scheme. Not all milestone builds are released at the same time so if the build you're looking for isn't here yet, DON'T PANIC. Check back later. If the source hasn't been released yet, you can always pull the source using cvs.
