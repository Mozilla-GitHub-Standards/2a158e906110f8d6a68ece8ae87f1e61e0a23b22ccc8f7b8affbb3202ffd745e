  • webclient is a thin Java wrapper around a native web browser, initially mozilla, and with support following later for IE and other browsers

  • webclient will be available as a Java Bean, as a Jar file, or as a Star1 component

  • the webclient project homepage is: http://www.mozilla.org/projects/blackwood/webclient/
