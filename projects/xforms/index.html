Mozilla XForms

Welcome to the Mozilla XForms Project. The goal of this project is to implement the W3C XForms 1.1 Candidate recommendation in Mozilla as an extension. The extension aims to work with all Mozilla projects building on the Mozilla platform, like Firefox, SeaMonkey, XULRunner, etc.

What Is XForms

In short, XForms is the new online forms standard envisioned by the W3C. More information about XForms, resources, articles, etc. can be found at our documentation page

Latest Release

Information on how to obtain the latest release of the Mozilla XForms extension can be found at our download page.
