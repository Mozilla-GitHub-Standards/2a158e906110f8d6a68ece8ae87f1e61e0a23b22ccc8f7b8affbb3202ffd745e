Mozilla Help Viewer Project

Welcome to the Mozilla Help Viewer project page. This page describes the development of a basic help viewer in Mozilla and its use as a browser for Mozilla end-user documentation. The viewer includes a simple facility for displaying context-sensitive help, uses RDF to load sets of topics dynamically, and provides navigation functionality for the HTML document it manages.

Firefox Help is part of Firefox since the 0.9 release. Thunderbird Help is still in an early stage. Please help!

Help System Features:

  • Built-in documentation search.
  • Support for retrieving Help documentation off of the Internet.
  • Customizable toolbars
  • Content for Firefox, Seamonkey and Thunderbird.

Interested in contributing?

See How You Can Help for more information about contributing. If you don't want to help at the code or documentation level, feel free to help by testing Help in nightly builds. On the non-writing and coding side, we currently need help with:

  • Beta testers and general QA work
  • Writing documentation, specifically for Thunderbird.
