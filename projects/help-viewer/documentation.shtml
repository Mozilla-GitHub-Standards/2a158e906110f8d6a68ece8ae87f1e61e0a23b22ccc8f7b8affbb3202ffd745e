Thunderbird

As of yet Thunderbird does not have working Help, although we are working at writing documentation and making the viewer work in Thunderbird. Because the documents used in the few early versions of Thunderbird Help are old and far from complete, we are working on an entirely new set of help docs. For now, the help docs can be viewed on the Mozilla wiki in the Thunderbird Help Documentation section. When the viewer is sufficiently ready and the docs have reached some sort of state of goodness, they will be added to the Mozilla source code.
