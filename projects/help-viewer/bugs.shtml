Bugs

Found a bug in Help?

If you find a bug in Mozilla Help Viewer, we encourage you to file a bug report to inform us about the problem.

Firefox Help

  • View all open Firefox Help Documentation bugs and make sure that your bug hasn't already been reported. If your bug has already been reported, you can add yourself to the CC list of that bug to see its progress.
  • Couldn't find a duplicate report? Then file a bug report. Make sure you read what to include in your report.

Seamonkey Help

  • View all open Seamonkey Help Documentation bugs and make sure that your bug hasn't already been reported. If your bug has already been reported, you can add yourself to the CC list of that bug to see its progress.
  • Couldn't find a duplicate report? Then file a bug report. Make sure you read what to include in your report.

What to include in a bug report:

  • A description of your problem.
  • Firefox/Seamonkey version number.
  • Steps to reproduce
  • What should happen instead of the problem reported.

Interested in fixing bugs?

If you are interested in fixing bugs, please take a look at How You Can Help. It should get you up and started pretty quickly with some decent starting instructions.
