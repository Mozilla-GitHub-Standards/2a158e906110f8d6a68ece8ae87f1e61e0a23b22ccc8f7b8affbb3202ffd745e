Phoenix Extensions

Supported Extensions for Phoenix 0.2

Below is a list of extensions that are guaranteed to work with Phoenix. These extensions were developed by third parties for use with the Phoenix browser. The links below point to the latest versions of these extensions.

Additional extensions may be found at MozDev.Org.

  • Mouse Gestures - This extension adds support for mouse gestures to Phoenix. Execute common commands like back, forward, and new window just by making the appropriate gesture with your mouse. [Developer's Site]

  • Preferences Toolbar - This extension adds support for an extra preferences toolbar to Phoenix. Access and manipulate common preferences directly from the toolbar. [Developer's Site]

  • Tabbed Browsing Extensions - This extension adds many features to tabbed browsing including tab reordering, the ability to control the positions and close behavior of tabs, and the ability to fix tabs. [Developer's Site]
