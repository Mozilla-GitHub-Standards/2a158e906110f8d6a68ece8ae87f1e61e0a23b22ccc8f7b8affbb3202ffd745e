Welcome to Camino®!

Thank you for downloading Camino®. We hope you will enjoy using this product. The latest version is Camino 1.5.5 and can be downloaded here.

You can give us Feedback—what do you like, and what don't you like?

See the Release Notes for information about features, and known issues in the current version.

For information about the Camino project, which produces the Navigator browser, see the Camino project page. Camino is a fully open-source project; to get involved in development, see the page for developers.

Plugin Warning

You are using a version of the Shockwave Flash plugin that has compatibility issues with Navigator. This can cause Navigator to crash on some pages. We strongly encourage you to download the latest Shockwave Flash plugin.
