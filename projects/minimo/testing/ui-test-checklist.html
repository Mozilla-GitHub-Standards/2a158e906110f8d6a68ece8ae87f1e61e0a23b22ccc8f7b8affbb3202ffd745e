Testing Minimo

Pre Requisites:

  • In WinCE, delete XUL cache and other files in /Windows/mozilla/minimo/

User Interface Test Checklist:

Initial Browser Load, Main UI

  • Check Homebar ( on the left ). Expect more than 2 icons on it;
  • Check if Default homepage was loaded;
  • Check Menu Display; Click bottom Left Menu and Right Menu, they should be exclusive;
  • Click URLBAR, check if brings Software Keyboard up;
  • Click Homebar, expect it to be collpased ( a few pixels width ). Expand it again with another click on it;

When Loading a Web Page

  • Check for the presence of a blue Progress Bar under the URLBAR when download is in progress. Test with a heavy page ( www.cnn.com ) to follow progress;
  • Check for the presence of the Status Bar above the URLBAR;
  • Check RSS alert icon. Go to slashdot.org, expect the URLBAR to display an RSS icon;

Page Contextual Features

  • Copy and Paste. Pen-select content, paste to text fiels. Copy From and Paste To, Content and URLBAR.
  • Main Menu -> Find. Expect the Find Toolbar to show below the URLBAR. Type a word ( existing on the page ) in it and click Next to get to it. Expect the string on the page to be selected. Click on Next should lead to next entry if available on the page.

RSS Loading

  • Expect: when viewing RSS page, it should display a basic list with the main title, and each item's title. Clicking on each item should simply load the page.
  • Check RSS 0.91, type rss:http://media-cyber.law.harvard.edu/blogs/gems/tech/sampleRss091.xml
  • Check RSS 2.0, type rss:http://rss.cnn.com/rss/cnn_topstories.rss
  • Check RSS 1.0, typerss:http://rss.slashdot.org/Slashdot/slashdot
  • Expect the Page Contextual Features ( above test item ) to work. Test Find, Copy, Paste.

Toggle Full Screen

  • Hold pen on page -> Context Menu -> select Full Screen; Expect Homebar to be available ( collpased or expanded states );
  • Main Menu -> Full Screen Mode. Expect the same.

Tabs

  • Launch multiple tabs. Click on Homebar icons to launch them ( Homebar should collapse each time you click ). Expect all the tabs to load and display favicons if available;
  • Select Main Menu -> New Tab. Expect a new empty Tab;
  • Access the Main Menu -> Browser Tabs -> select a Tab via the keyboard. Expect browser to show the selected Tab;
  • Check Toggle between tabs with Mouse click;
  • Check Tab Contextual Menu. Choose a Tab, Hold the pen on top of it. Expect a context menu. Test the functions.

File Download

  • Test download with no Default Directory set. Should display a dialog with a request to select a folder. Select a folder and expect the download toolbar to show below to the URLBAR. When the download is complete, expect a revel location button to replace the Cancel button.
  • Main Menu -> Preferences -> Network ( globe icon ) -> Set Save Directory, select a folder. With this setting, expect the next download to *not* display a dialog and simply initiate the download progress.

Preferences General UI

  • Check alternate between panels, click on each panel on top;
  • Preferences -> General: Change default homepage, restart Minimo, expect to be displayed when it launches;

SSR ( Small Screen Rendering )

  • By default SSR is on. Page content should fit in window width / page height may grow.
  • Preferences -> General. Uncheck SSR. Save preferences with [OK] button. Go to page and reload. Expect the page to be loaded in its normal form.

Bookmarks

  • Nav Menu ( bottom right ) -> Bookmarks. Should show empty bookmark message if Bookmark is empty.
  • Add a bookmark. Go to a page, Nav Menu -> Bookmark This Page. Nav -> Menu -> Bookmarks, expect the page entry to be in.
  • Check delete Bookmarks. Preferences -> Privacy -> Delete All Bookmarks. Go to item 1, and check if Bookmark is empty.

URLBAR History

  • For each page you access, the History Manager should store it in its list. Type a page URL on the URLBAR and load it. Then go to URLBAR and initiate typing again. Expect the History Dropdown to show. Select the same page in the list, and expect the page to load.
