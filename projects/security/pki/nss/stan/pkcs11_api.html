Module Management Interface

NSS_GetLoadedModules
Return a NULL-terminated array of all loaded PKCS #11 modules. 
NSSModule **
NSS_GetLoadedModules (
  void
);

NSS_LoadModule
Load a PKCS #11 module from a library file. 
NSSModule *
NSS_LoadModule (
  NSSUTF8 *moduleOpt,
  NSSUTF8 *uriOpt,
  NSSUTF8 *opaqueOpt,
  PRBool keepModule,
  void *reserved
);

NSS_UnloadModule
Unload the specified PKCS #11 module. 
PRStatus
NSS_UnloadModule (
  NSSModule *module
);

  NSSUTF8 *name

  NSSSlot *slot

Set the password on the device. This initializes the slot if needed, or changes the password if one has already been set.

  NSSSlot *slot,

  NSSUTF8 *newPassword
);
