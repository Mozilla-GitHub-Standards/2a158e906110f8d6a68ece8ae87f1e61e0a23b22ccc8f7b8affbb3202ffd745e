  • Restart runs utilize cache, full runs handshake every connection ("strsclnt -N").
  • Stressing a 4 cpu Solaris 8 machine requires 7 client machines.
  • Each strsclnt is run with one hundread threads ("strsclnt -t 100").
  • The SSL_RSA_WITH_RC4_128_MD5 (SSL3) cipher is used ("strsclnt -C c").
  • Nagle delays in TCP are disabled on the client ("strsclnt -D").
  • All client machines run under latest RTM version of NSS.
