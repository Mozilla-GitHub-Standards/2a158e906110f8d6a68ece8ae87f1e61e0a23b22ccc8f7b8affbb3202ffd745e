Interactive Leak Tests

Here's a list of some interactive tests that I've been running while searching for leaks. Bytes leaked represents the total number of bytes leaked as shown using the trace ref count logs. The majority of these tests don't leak enough to warrant a bug. The big ones do have bug numbers. All numbers are for win32.

You'll need a debug build to perform these tests.
Set XPCOM_MEM_LEAK_LOG=filename.
Start mozilla.
Run the test.
Exit mozilla.
Open filename to view the results.

Details to run the tests are at Leak test how to
