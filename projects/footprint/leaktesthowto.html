For each of these tests set XPCOM_MEM_LEAK_LOG=filename. Then start mozilla, run the test and exit. Save any unsaved files. Then open filename to view the results.

Browser tests
Find text in file
  • Go to mozilla/layout/base/nsPresShell.cpp
  • Control-F to open the find dialog
  • Enter 7000
  • Click Find
Open New Window
  • Go to mozilla/layout/base/nsPresShell.cpp
  • Control-N to open a new window
  • In the new window, Go to mozilla/layout/base/nsPresShell.cpp
Session History
  • Go to mozilla/layout/base/nsPresShell.cpp
  • Go to www.cnn.com
  • Click Back
  • Click Forward
  • Click Back
  • Click Forward
Change Charset
  • Go to mozilla/layout/base/nsPresShell.cpp
  • View -> Character Coding -> Western (Windows-1252)
  • Click Reload
Save As
  • Go to mozilla/layout/base/nsPresShell.cpp
  • File -> Save As
  • Enter a filename
  • Click Save
Send Page
  • Go to mozilla/layout/base/nsPresShell.cpp
  • File -> Send Page
  • Enter a recipient
  • Click Send
Send Link
  • Go to mozilla/layout/base/nsPresShell.cpp
  • File -> Send Link
  • Enter a recipient
  • Click Send
Print
  • Go to www.cnn.com
  • File -> Print
  • Click Ok
Select / Select All
  • Go to mozilla/layout/base/nsPresShell.cpp
  • Control - A to select all
  • Click in the page to deselect
  • Click and drag a few times to select various sections of the text
  • Click, and drag down until the page scrolls while selecting
Show/Hide Nav Bar
  • View -> Show/Hide -> Navigation Toolbar
Show/Hide Status Bar
  • View -> Show/Hide -> Status Bar
Show/Hide Personal Toolbar
  • View -> Show/Hide -> Personal Toolbar
Show/Hide Sidebar
  • View -> Personal Sidebar
View Source
  • Go to mozilla/layout/base/nsPresShell.cpp
  • View -> Page Source
Shutdown
  • Start and Exit

Bookmarks
Open/Close Bookmark Manager
  • Control-B to open Bookmark Manager
Sort
  • Control-B to open Bookmark Manager
  • Open a few folders to get a long list of bookmarks
  • View -> Sort by Name
  • View -> Z>A Sort Order
  • View -> Sort by Location
  • View -> A>Z Sort Order
  • View -> View -> Sort by Keyword
  • View -> Z>A Sort Order
Add/Remove
  • Go to www.cnn.com
  • Bookmarks -> Add Bookmark
  • Repeat 10 times
Composer (start mozilla with -edit)
Change views
  • File -> Open File
  • Select a file (I went with the saved copy of nsPresShell.cpp)
  • Click Open
  • Click the Show All Tags tab
  • Click the <HTML> Source tab
  • Click the Preview tab
  • Click the Normal tab
Add lots of images
  • Click the Image button
  • Click Choose File
  • Click Open
  • Enter some Alternate Text
  • Click OK
  • Click to select the image
  • Control-C to copy the image
  • Control-V 10 times to paste copies of the image
  • Select all copies of the image
  • Control-C to copy the images
  • Control-V 10 times to paste a bunch of images

add lots of tables
  • Click the Table button
  • set 4 rows
  • set 4 columns
  • Click OK
  • insert 2 or 3 characters into each cell
  • Control-A to select everything
  • Control-C to copy the table
  • Control-V 10 times to paste copies of the table
add links
  • Click the Link button
  • Enter text for the link
  • Set the web location to www.cnn.com
  • Click OK
  • Select the link
  • Control-C to copy the link
  • Control-V 10 times to paste copies of the link
delete lots of items (start without -edit)
  • Go to www.cnn.com
  • File->Edit Page
  • Select an image and press delete
  • Select select objects at once and press delete
  • Select some text and type 'foo' to replace it
cancel without saving
  • repeat the delete lots of items test, don't save when you exit ;)

Mail News (start mozilla with -mail)
Cancel Message without save
  • Control-M to create a new mail message
  • Add a recipient
  • Add a subject
  • Add some text to the body
  • File->Close
  • Click Don't Save
Cancel Message with Save
  • Control-M to create a new mail message
  • Add a recipient
  • Add a subject
  • Add some text to the body
  • File->Close
  • Click Save
Send a message
  • Control-M to create a new mail message
  • Add a recipient
  • Add a subject
  • Add some text to the body
  • Click Send
Open message in new window
  • Open a folder of messages
  • Double-Click a message subject to bring the message into a new window
  • Repeat 10 times
Search Messages (for this test I use netscape.public.mozilla.porkjockeys)
  • Open a newsgroup
  • Search-> Search Mail/News Messages
  • Set the search criteria to Subject Contains 'mem'
  • Sort the results by subject
create/remove a filter
  • Edit->Message Filters...
  • Click New
  • Enter a filter name
  • Set the criteria to Subject Contains 'mem'
  • Select a destination folder
  • Click OK
Sort messages (for this test I use netscape.public.mozilla.porkjockeys)
  • Open a newsgroup
  • View -> Sort By -> Subject
  • View -> Sort By -> Date
  • View -> Sort By -> Sender
  • View -> Sort By -> Ascending
  • View -> Sort By -> Thread

delete a message
  • Select a message
  • Right-Click -> Delete
  • Repeat 10 times
forward a message
  • Select a message
  • Right-Click -> Forward
  • Enter a recipient
  • Click Send
attach something to a message
  • Control-M to start a new message
  • Enter a recipient and subject
  • Click the Attachment panel
  • Select a file and click Open
  • Click Send
shutdown
  • Start and exit

General

  • Bring the mouse over every element in the UI, pause to bring up tooltips
change focus
  • Go to www.cnn.com

  • In the new window, go to www.mozilla.org
  • Alt-Tab 10 times to change focus

  • go to www.mozilla.org
  • Click the minimize button
  • Restore the window
  • Click the maximize window
  • repeat
change themes

  • View->Change Themes->Classic

  • Click Submit query
