XUL’s Advantages

  • Scriptable. Changing UI without recompilation. Application development cycle is dramatically reduced.

    •  
  • Cross-platform. The same set of XUL files are portable across platforms including Unix, Mac, and Windows 32.

    •  
  • Customizable. With the introduction of "Configurable Chrome", all components of the chrome can be configured either statically in the chrome registry or dynamically on the fly.
