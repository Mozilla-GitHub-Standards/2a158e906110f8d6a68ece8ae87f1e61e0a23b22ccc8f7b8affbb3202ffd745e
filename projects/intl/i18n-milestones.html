    For a Windows-xxx reference, see Codepage & Co. and Microsoft's:

    • Character sets and codepages
    • Windows codepages
    • OEM codepages
    For a reference on Mac OS encodings, see the info posted to unicode.org or the same info posted to apple.com.

    For an explanation about ISO-8859-6-i, ISO-8859-6-e, ISO-8859-8-i, and ISO-8859-8-e, see RFC1556.

    For the recognized charset names used for the HTTP charset or the META tags, see the source for charsetalias.properties.  For the list of charsets assigned by IANA, see the IANA CHARACTER SETS registry.
