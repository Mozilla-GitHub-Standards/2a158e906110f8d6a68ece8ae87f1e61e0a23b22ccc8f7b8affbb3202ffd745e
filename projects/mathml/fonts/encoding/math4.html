Math4 Font

Math4 Encoding

Font-family: Math4
Character Map:

 Screenshot, and Encoding result generated by encode.pl
Copyright: © 1996, 1997 Wolfram Research. Data © 1996, 1997 Glenda de Guzman.
Availability:  
Issues:
  • There are seemingly redundant glyphs (e.g., 0x43 and 0x55, 0x45 and 0x64, the 'hooks') but these have different spacing/lbearing. For some lbearing = 0, while lbearing > 0 for others. We retained those with lbearing > 0 which might be better on the web/screen.
  • Pieces intended for arrows with barbs have a subtle spacing depending on the side of the barb (left/right/up/down).
  • Many kinds of extensible arrows can be defined by combining the parts. The MathFont Property File is hand-tuned to include additional combinations not listed here.
