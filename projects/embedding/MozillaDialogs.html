Mozilla Dilaog List

The following table lists the dialogs which are displayed by Mozilla. Please note that this list includes only the the dialogs which are displayed via the nsIPrompService interface, such as, Alerts, Prompts etc. "Regular" dialogs such as the Preferences dialogs etc. are currently not included in this list and will be added to it at a later date, if needed.

While i've made all efforts to make this list as complete as possible i may have missed(or mis-represented) some of them. Please let me know and i'll make the necessary changes.


Dialog Message
When Displayed?
Comments/Notes
Wallet
Alerts
Saving Passwords and Other Sensitive Information. Password Manager and Form Manager will save passwords, user names, and other sensitive information and enter them for you automatically when they are required. This sensitive information is stored on your computer in a file that's difficult, but not impossible, to read.If other people have access to your computer, you may want to password protect the stored sensitive information by choosing a Master Password.If you choose to password protect your stored information, you will be asked to provide your Master Password from time to time.  This approach provides better security but is slightly less convenient.
 Displayed if this is the first time the data is being captured by the Wallet component.

Unable to convert stored data.
 Displayed when the wallet component fails to encrypt/decrypt the stored data
There are no fields that can be prefilled.
 Displayed when wallet fails  to prefill a form.
Data has been captured.
 Displayed when wallet has successfully captured data.
There is nothing to capture.
 Displayed when there's nothing for wallet to capture .
Unable to capture data.
 Displayed when wallet cannot capture any data - due to an encryption failure
Unable to change the password that protects your sensitive information.
 Displayed when the user chooses to change the "Change Master Password.." menu item and for some reason it is unable to change the password.
Your sensitive information is now locked.
 Displayed when???????
Unable to lock your sensitive information.
 Displayed when???????



Confirmation Dialogs
Password Manager can remember this logon and enter it automatically the next time you return to this website.Do you want Password Manager to remember this logon?
 Displayed by the password manager when a user fills in and submits a login form.
  • This dialog will give the user the following three choices(via push buttons) : Yes, No, Never For This Site
Prompts
Password:
 Displayed when the Wallet component needs a password to be entered by the user.
  • The dialog will also have a checkbox titled "Use Password Manager to remember this password" - which when checked saves the password.
Prompt:
 Displayed when the wallet component needs to prompt the user for some value.
  • The dialog will also have a checkbox titled "Use Password Manager to remember this value"  - which when checked saves the value entered.
UserName:

Password:
Displayed when the wallet component needs to prompt the user for a username/password.
  • The dialog will also have a checkbox titled "Use Password Manager to remember these values" - which when checked saves the values entered.
Caps(Script Security Manager)
Confirmation Dialogs
A script from <HostName> has requested enhanced privileges. You should grant these privileges only if you are comfortable downloading and executing a program from this source. Do you wish to allow these privileges?
 Displayed when the script from the <HostName> is trying to perform some privileged operation
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision
Docshell
Alerts
<FileName> does not exist or could not be accessed. Please check the file path and try again.
 Displayed by DocShell when a document it's trying to load fails due to the file not being found.
<HostName> could not be found. Please check the name and try again.
 Displayed by DocShell when a document it's trying to load fails because the host was not found.
The connection was refused when attempting to contact <HostName>.
 Displayed by DocShell when a document it's trying to load fails because it could not connect to the server.
The operation timed out when attempting to contact <HostName>.
 Displayed by DocShell when a document it's trying to load fails because of the socket funtions' timeout.
<Protocol> is not a registered protocol.
 Displayed by DocShell when it's trying to load a URI and it's unable to find a protocol handler for <Protocol>.
Access to the port number given has been disabled for security reasons.
 Displayed by DocShell when it's trying to open a URI and gets an error from the server saying that access on that port number is not allowed.
Confirmation Dialogs
The page you are trying to view contains POSTDATA that has expired from cache. If you resend the data, any action the form carried out (such as a search or online purchase) will be repeated. To resend the data, click OK. Otherwise, click Cancel.
 Displayed when saying the page has expired from cache and to ask the user if they wished to refetch the page.
The page you are trying to view contains POSTDATA. If you resend the data, any action the form carried out (such as a search or online purchase) will be repeated. To resend the data, click OK. Otherwise, click Cancel.
 Displayed when there is a valid postdata *and* the user presses reload or shift-reload. This is to get the user's permission before reposting the data to the server.
Turbo Mode[Windows Only]
Alerts
Quick Launch is now disabled. To enable it again, open the Edit menu and choose Preferences. Choose the "Advanced" category and check "Enable Quick Launch.
 Displayed when the user chooses the "Disable Quick Launch" menu item from the TrayIcon.
Cookies
Confirmation Dialogs
The site <HostName> wants to modify an existing cookie.  Do you want to allow it?
 Displayed when a previous cookie with that name is already present and the host is trying to modify the existing cookie.
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision
The site <HostName> wants permission to set another cookie.  You already have <N> cookies from this site.  Do you want to allow it?
 Displayed when the user already has N number of cookies from a host named <HostName> and the host is trying to set another cookie
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision
The site <HostName> wants to set a second cookie.  Do you want to allow it?
 Displayed when the host named <HostName> has already set one cookie before and is now trying to set a second cookie.
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision
The site <HostName> wants to set a cookie.  Do you want to allow it?
Displayed when the host named <HostName> wants to set a cookie.
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision
The site <HostName> wants to load an image.  Do you want to allow it?
 Displayed when "image blocker" pref is enabled and we're trying to load an inage from the host named <HostName>
  • This dialog is a YES/NO type dialog and the dialog also has a check box with the message "Remember this decision" which when checked remembers the user's decision

Profile Manager
Alerts
This profile cannot be migrated because the directory containing the profile information could not be found. Choose another profile or create a new one.
 Displayed when the profile manager can't find the source dir from which it's trying to migrate an old profile from.
A profile with this name already exists. Please choose another name
 Displayed when the user chooses to rename a profile and a profile with the same name already exists.
Confirmation Dialogs
You are about to create a <BrandName>profile based on your Netscape 4 profile. This will copy and convert some of your Netscape 4 files to work with <BrandName>.  Your Netscape 4 profile data will not be modified.
 Displayed when Mozilla/Netscape is run from the first time on a machine with a previous installation of Netscape 4.x.
  • This dialog will also have two buttons with the following titles : "Convert Profile" and "Manage Profiles"
The profile you selected was created with a previous version of Netscape and must be copied and converted to a <BrandName> profile before it can be renamed. Do you want to convert this profile?
 Displayed when the user chooses to rename a profile which was created with an earlier version of Netscape.
  • This is a Yes/No type dialog.
The profile you selected was created with a previous version of Netscape. <BrandName> can remove this profile from the list of profiles that it maintains but you must remove it from your system using the version of Netscape that created it, or by hand. Do you want to remove this profile from the list of profiles?
 Displayed when the user chooses to remove a profile which was created with an earlier version of Netscape
  • This is a Yes/No type dialog.
Deleting a profile will remove the profile from the list of available profiles and cannot be undone.You may also choose to delete the profile data files, including your saved mail, settings, and certificates. This option will delete the folder <FolderName> and cannot be undone. Would you like to delete the profile data files?
 Displayed when a user chooses to delete a profile
  • The dialog will contain the following three buttons : "Don't Delete Files", "Delete Files", "Cancel"
The profile you selected was created with a previous version of Netscape and must be copied and converted before you can use it with <BrandName>. Your original 4.x profile will not be altered. Do you want to copy and convert this profile?
 Displayed when the user chooses to start the <BrandName> client with a profile which was created with an earlier version of Netscape
  • This is a Yes/No type dialog.
Plugins
Alerts
Netscape cannot find the Plugin Downloader Plugin.  Without the Plugin Downloader Plugin, you cannot automatically download and install plugins.  Please visit http://www.netscape.com/ to install the Plugin Downloader Plugin. When "default plugin" is not found on the user's system .
 When the user visits a page with a plugin and the plugin for that content type is not installed, Netscape uses the "default plugin" to download the plugin from Netscape's plugin-finder page. This dialog is shown when the "default plugin" is missing (i.e. the user removed it) from the user's Netscape distribution
  • This dialog also will have a check box with the following text:
      • "I know I need the Plugin Downloader Plugin, but don't show me this dialog again."
    which, when checked, will not display this dialog again.
The plugin performed an illegal operation. You are strongly advised to restart Navigator.
 This dialog is displayed when plugin throws unhandled exception.
  • This dialog also will have a check box with the following text:
      • "Don't show this message again during this session"
    which, when checked, will not display this dialog again during this session.
DOM
Confirmation Dialogs
A script on this page is causing mozilla to run slowly. If it continues to run, your computer may become unresponsive. Do you want to abort the script?
Not really sure under what circumstances this gets shown
  • This string is hardcoded in the code. Needs to moved into a string bundle
Bookmark Service
Confirmation Dialogs
The following web page has been updated. Title:<Page Title Here> URL:<Page's URL here>. Would you like to display it?
 Displayed when a page the user has scheduled for update notification is updated.
  • This is a Yes/No type dialog and the dialog also will also have a checkbox titled "Stop checking for updates on this web page". When checked the page will not be checked for updates anymore.
FTP Protocol Handler
Prompts
Enter username and password for <HostName>
 Displayed when the FTP host requires a username/password to login to it
  • The dialog will also have a checkbox which when checked remembers the username and password just entered
Enter password for <UserName> on <HostName>
 Displayed when the FTP host requires a password for the given <UserName>
  • The dialog will also have a checkbox which when checked remembers the password just entered
HTTP Protocol Handler
Prompts
Enter username and password for proxy at <ProxyServerName>
 Displayed when the user needs to login to the proxy server
  • The dialog will also have a checkbox which when checked remembers the username and password just entered
Enter username and password for <Realm> at <HostName>
Displayed when the user needs to be authenticated to access <Realm> at <HostName>
  • The dialog will also have a checkbox which when checked remembers the username and password just entered

 Displayed when the user chooses to save a file and a file with that name already exists

Sidebar Component
Alerts
<URL> already exists in the <PanelName>
Displayed when the user tries to add a URL to a sidebar panel and that <URL> already exists

 Displayed when the user chooses to apply a new theme
