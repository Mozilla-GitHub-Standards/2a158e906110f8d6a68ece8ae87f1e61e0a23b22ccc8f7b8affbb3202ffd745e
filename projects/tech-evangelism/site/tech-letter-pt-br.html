Usando padrões da internet nas suas páginas
-------------------------------------------
Alguns elementos e práticas consideradas "padrão" entre desenvolvedores web para adicionar DHTML e/ou compatibilidade entre navegadores para suas páginas foram excluídas das especificações HTML 4.01 e Document Object Model (DOM) do W3C. Descubra o que é suportado.
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages

O Document Object Model (DOM) no Mozilla
----------------------------------------
http://www.mozilla.org/docs/dom/ contém links úteis para aprender e usar o Document Object Model (DOM) no Firefox e outros Navegadores baseados no Gecko. A referência do DOM para o Gecko
http://developer.mozilla.org/en/docs/Gecko_DOM_Reference
dá para o desenvolvedor web iniciante um guia sobre a implementação W3C DOM do Firefox e inclui vários exemplos.

Detecção de navegadores e suporte entre navegadores
---------------------------------------------------
Quando você detecta navegadores com o renderizador Gecko, é mais simples para você procurar por "Gecko" ao invés de um nome de produto específico ou o a string de user agent completa. Você vai achar mais detalhes neste artigo:
http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages#Developing_cross_browser.2Fcross_platform_pages

-----------------------------------------------
Venkman é um completo debugger interativo para Firefox e Seamonkey que roda em qualquer plataforma onde o Firefox e o Seamonkey são suportados. Para uma introdução leia:
http://developer.mozilla.org/en/docs/Venkman
