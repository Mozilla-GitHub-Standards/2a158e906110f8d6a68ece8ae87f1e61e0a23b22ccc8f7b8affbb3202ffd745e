Utilisation des standards du Web dans vos pages
-----------------------------------------------
Certains éléments et pratiques considérés comme "standard" parmi les développeurs web pour le DHTML et/ou le support multi-navigateurs ne font plus partie des spécifications HTML 4.01 et Document Object Model du W3C. Vérifiez ce qui est supporté.
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages

Détection du navigateur et support multi-navigateurs
----------------------------------------------------
Si vous désirez détecter les navigateurs utilisant Gecko, le plus simple est de rechercher le terme "Gecko" dans la chaîne User Agent plutôt qu'une marque spécifique de navigateur ou une version particulière
Pour plus d'information:
http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages#Developing_cross_browser.2Fcross_platform_pages

---------------------------------------------
Venkman est un débogueur interactif et riche en fonctionnalités pour Firefox et Seamonkey. Il fonctionne sur toutes les plateformes pour lesquelles Firefox et Seamonkey sont disponibles.
Une introduction est disponible ici:
http://developer.mozilla.org/fr/docs/Venkman
