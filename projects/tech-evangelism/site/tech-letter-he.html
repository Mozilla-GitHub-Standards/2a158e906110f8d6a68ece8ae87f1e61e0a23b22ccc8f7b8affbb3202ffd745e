לכבוד בוני האתר,התאמה: במידה שהשם ידוע, כדאי להוסיף אותו ולשנות את מין הפניה של המכתב בהתאם

אתרכם התאמה: הכניסו כאן כתובת אינו תומך בדפדפן פיירפוקס 2 או בדפדפנים אחרים שמבוססים על מנוע התצוגה Gecko כדוגמת קמינו 1.5, סימונקי 1, אפיפני 2, גלאון 2 וקיי-מלאון 1.

דו"ח בנושא הזה מופיע במערכת הבאגים של mozilla.org. אתם יכולים לצפות בתגובות ולהוסיף תגובה משלכם:

https://bugzilla.mozilla.org/show_bug.cgi?id=התאמה: מספר באג

התאמה: הוסיפו כאן תיאור טכני של הבעיות, והמלצות כיצד ניתן לתקן אותן. תוכלו גם לצרף דוגמה שמציגה כיצד ניתן לשנות את קוד האתר כך שיתמוך בתקנים ובפיירפוקס ודפדפנים אחרים מבית מוזילה.

כדי להבטיח שאתרכם יעבוד כראוי עם דפדפני הדור־הבא האלה, כדאי שתקפידו על ההמלצות של ECMA לגבי ECMAScript (הידועה כ־JavaScript) ועל ההמלצות של ארגון ה־W3C לגבי HTML 4.01, CSS 2.1 וה־DOM (Document Object Model). הדבר יבטיח גם תאימות לדפדפנים אחרים שתומכים בתקנים פתוחים כדוגמת אופרה 9.5, ספארי 2 ואינטרנט אקספלורר 7.

מקורות המידע הבאים יעזרו לכם לתמוך בתקנים פתוחים, תוך שמירה על תאימות מלאה לאינטרנט אקספלורר של מיקרוסופט. חשוב גם להכיר את נושא זיהוי דפדפנים מבוססי Gecko, כפי שמתואר למטה.

שימוש בתקני האינטרנט בעמודי האינטרנט שלכם
-----------------------------------------
מספר אלמנטים ודרכים שמקובלות בקרב בוני אתרים להוספת DHTML ולהוספת תמיכה לדפדפנים שונים לא נכללו בתקן HTML 4.01 של ה־W3C ובהגדרות ה־Document Object Model, זאת אף שאלמנטים אלו עשויים להחשב כ־"תקניים".
בדף הבא ניתן למצוא הסבר על תהליך שדרוג התכנים של עמודי הוו'ב שלכם, כך שיתמכו בתקנים של ה-W3C:‏
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages

Document Object Model בפיירפוקס
-------------------------------
בעמוד http://www.mozilla.org/docs/dom ניתן למצוא קישורים מועילים ללימוד ושימוש ב־ Document Object Model (DOM)‎ בפיירפוקס ובדפדפנים אחרים שמבוססים על Gecko.

תיעוד בנושא http://developer.mozilla.org/en/docs/Gecko_DOM_Reference Gecko DOM מספק לבוני אתרים מתחילים מדריך ליישום של ה־W3C DOM בפיירפוקס וכולל דוגמאות רבות.

---------------

http://developer.mozilla.org/en/docs/Web_Standards

זיהוי הדפדפן ותמיכה בדפדפנים שונים
----------------------------------
כדי לזהות דפדפן שעושה שימוש במנוע ה־Gecko של נטסקייפ, קל יותר לחפש את המחרוזת "Gecko", מאשר לסרוק אחר שם מסויים של חברה ומחרוזת הזיהוי המלאה של הדפדפן. ניתן למצוא פרטים נוספים במאמר הבא:
http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages:Developing_Cross_Browser/Cross_Platform_Pages
