Validar sus documentos HTML & CSS
.................................

El W3C pone a su disposición un validador en 
http://validator.w3.org/ 
que le puede ayudar a convertir el código de sus páginas en HTML 
válido, que será mostrado correctamente por cualquier navagador que 
respete los estándares. El W3C también proporciona una serie de 
tests CSS 2.1 en 
http://www.w3c.org/Style/CSS/Test/ 
y un validador CSS 2.1 en español en 
http://jigsaw.w3.org/css-validator/validator.html.es 
Tenga en cuenta que el analyzador CSS de Mozilla es bastante estricto y 
que un código CSS no válido aceptado por otros navegadores como Internet 
Explorer 7 puede no ser aceptado por Mozilla. Los navegadores basados en 
Gecko interpretarán correctamente cualquier código CSS que sea 
reconocido correcto por el validador.

Artículos
.........

Hay muchos artículos que pueden ayudarle a hacer la transición hacia 
el soporte de los estándares:

Se ha dedicado una sección especial de mozilla.org a Desarrollares Web
como usted. Consúltela en:
http://developer.mozilla.org/docs/Using_Web_Standards_in_your_Web_Pages
http://developer.mozilla.org/es/docs/Desarrollo_Web

Para una introducción al desarrollo web "cross-browser", visite:
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages:Developing_Cross_Browser/Cross_Platform_Pages

El API DHTML, que es un cuerpo de código abierto que define un 
conjunto de funciones para establecer propiedades de posicionamiento 
CSS compatibles con varios navegadores, ha sido actualizado para 
reflejar el soporte de los nuevos estándares que tienen los 
navegadores basados en Mozilla. Puede encontrarlo en la 
dirección siguiente:
http://developer.mozilla.org/es/docs/Est%C3%A1ndares_Web

Aprenda a cambiar las propiedades CSS dinámicamente usando 
Javascript "cross-browser" en:
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages:Using_the_W3C_DOM#Accessing_Elements_with_the_W3C_DOM

Esta página también proporciona un Generador de Código para generar 
Javascript "cross-browser" que le permitirá establecer propiedades 
CSS y que funciona en Internet Explorer 7, Opera 9.5, Safari 3 ys 
navegadores basados en Mozilla.

Mozilla también tiene un sitio dedicado a la promoción del desarrollo 
cross-browser para Firefox 2 y el resto de los navegadores basados en 
Mozilla, así como para Internet Explorer 7, en:
http://developer.mozilla.org/es/docs/Portada

Echele un vistazo al "Detector Práctico de Navegadores" que se 
encuentra en:
http://developer.mozilla.org/docs/Browser_Detection_and_Cross_Browser_Support

Aprenda a actualizar sus páginas DHTML para que funcionen en Firefox 2 
y Seamonkey en:
http://developer.mozilla.org/en/docs/Gecko_DOM_Reference

Aprenda a diagnosticar cúales son los problemas que impiden que sus 
páginas web se muestren correctamente en Firefox 2 y Seamonkey:
http://www.mozillaes.org/foros/viewforum.php?f=3 (MozillaES foro Tecnoevangelismo)
http://www.mozillaes.org/foros/viewforum.php?f=13 (Desarrollo web: HTML, CSS, 
JS, accesibilidad,...)

Recursos en castellano y catalán (entre ellos, traducciones de las 
especificaciones del W3C):
http://www.w3.org/2005/11/Translations/Lists/ListLang-es.html
http://html.conclase.net/
http://www.minid.net/utilidades/w3c/
http://www.tierradenomadas.com/
