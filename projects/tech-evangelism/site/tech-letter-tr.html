Siteniz [DEĞİŞTİR: Site adresi], Firefox 2 web tarayıcısını veya Gecko motoru üzerine kurulmuş Camino 1, Seamonkey 1, Epiphany 2, Galeon 2 ve K-meleon 1 gibi diğer tarayıcıları desteklememektedir.

Mozilla.org'un hata sisteminde bu konuyla ilgili bir rapor dosyası oluşturulmuştur. Siz de bu konuyla ilgili yorumları görebilir veya kendiniz de yorumda bulunabilirsiniz:
https://bugzilla.mozilla.org/show_bug.cgi?id=[DEĞİŞTİR: Hata numarası]

[DEĞİŞTİR: Sorunların teknik tanımlarını ve çözüm önerilerini buraya koyun. Ayrıca site içeriğinin web standartlarını ve Mozilla'yı desteklemesi için sitenin kodunun nasıl değiştirileceğini gösteren çalışan örnekleri de buraya koyabilirsiniz.]

Sitenizin içeriğinde ECMAScript'ler (JavaScript) hakkındaki ECMA tavsiyelerini ve HTML 4.01, CSS 2.1, ve DOM (Document Object Model) hakkında W3C tavsiyelerini uyguladığınız zaman siteniz bu yeni-nesil web tarayıcılarla çalışaktır, hem de Opera 9, Safari 2 ve Internet Explorer 7 gibi açık standartları destekleyen web tarayıcıları ile de çalışmaya devam edecektir.

Aşağıdaki kaynaklar sitenizi açık standartları destekler hale getirmenize yardımcı olacağı gibi sitenizin Microsoft Internet Explorer ile de çalışmaya devam etmesini sağlayacaktır. Gecko web tarayıcıları tespiti hakkındaki bilgilendirme ayrıca önemlidir.

--------------------------------------------------
http://www.ecma-international.org/publications/standards/Ecma-262.htm

World Wide Web Konsorsiyumu (W3C) Teknik Tavsiyeleri

W3C, Firefox'un kullandığı "World Wide Web" için evrensel standarları belirler.
http://www.w3.org/TR/ (english)
http://www.w3.org/2005/11/Translations/Lists/ListLang-tr.html (W3C dokümanlarının Türkçe çevirileri)

Mozilla Geliştirici Merkezi Dökümantasyonu
------------------------------------------
Firefox 2 ve diğer Gecko motoru tabanlı web tarayıcılarına özgü bilgilendirmeler için Mozilla Geliştirici Merkezini ziyaret edin.
http://developer.mozilla.org/en/docs/Main_Page (english)

Web Sayfalarınızda Web Standartlarını Kullanmak
-----------------------------------------------
Web geliştiricileri arasında içeriğe DHTML ve tarayıcılar arası destek ilave etmek için standart olarak kabul edilen çeşitli elemanlar ve uygulamalar W3C'nin HTML 4.01 ve "Document Object Model" spesifikasyonlarından çıkartılmıştır.
Web sayfalarınızın içeriğinin W3C'nin web standartlarına uygun hale gelmesi için gerçekleştirilecek yükseltme işleminin bir özeti aşağıdaki bağlantıda sunulmuştur.
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages (english)

Mozilla'da "Document Object Model"
--------------------------------------
http://www.mozilla.org/docs/dom/ , Mozilla'da ve diğer Gecko tabanlı tarayıcılarda "Document Object Model (DOM)" kullanımını öğrenmek için pek çok yararlı bağlantı içermektedir.

Gecko'nun DOM referansı http://developer.mozilla.org/en/docs/Gecko_DOM_Reference, sayısız örnekle birlikte Firefox'da W3C DOM uygulaması hakkında başlangıç seviyesinde bir web geliştirici rehberi içermektedir.

DHTML Sayfalarının Güncellenmesi
------------------------------------
document.all ve diğer bağıntılı uzantılar ve taraycılar arası dünyada bunlarla çalışma konusunda bilmek isteyeceğiniz her şey aşağıdaki bağlantıda sunulmuştur:
http://developer.mozilla.org/en/docs/Web_Standards (english)

Tarayıcı Tespiti ve Tarayıcılar Arası Destek
--------------------------------------------
Gecko üzerine kurulmuş web tarayıcılarını tespit ettiğinizde sizin için en kolayı, belirli bir dağıtıcı ismine veya kullanıcı ajanı bilgisine ulaşmaktan ziyade "Gecko" tanımına ulaşmanızdır. Bunun hakkında daha fazla bilgiyi aşağıdaki makalelerde bulabilirsiniz:
http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages#Developing_cross_browser.2Fcross_platform_pages (english)
