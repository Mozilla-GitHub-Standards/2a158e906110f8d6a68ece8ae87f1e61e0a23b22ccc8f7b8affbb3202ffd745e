Stosowanie standardów WWW na Państwa stronach
---------------------------------------------
Szereg elementów uważanych przez wielu autorów stron jako "standardowe" przy wprowadzeniu DHTML (dynamicznego HTML-u) i/albo tworzeniu stron współpracujących z różnymi przeglądarkami zostało wyłączone ze specyfikacji przyjętych przez W3C: HTML 4.01 oraz DOM (Document Object Model). Prosimy przekonać się co jest standardem.
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages

Document Object Model a przeglądarka Mozilla
------------------------------------------
http://www.mozilla.org/docs/dom/ zawiera przydatne odnośniki do dokumentów pomagających w nauczeniu się prawidlowego stosowania Document Object Model (DOM) w Firefoksie oraz innych przeglądarkach opartych na kodzie Gecko.

Dokumentacja Gecko DOM http://developer.mozilla.org/pl/docs/Dokumentacja_Gecko_DOM dostarcza początkującemu autorowi stron przewodnik po implementacji W3C DOM przez Firefoksa, przedstawiając wiele praktycznych przykładów.

---------------------------------------

http://developer.mozilla.org/pl/docs/Standardy_WWW

Wykrywanie przeglądarek
-----------------------
W celu wykrycia przeglądarki zawierającej kod Gecko najprościej szukać wewnątrz jej pełnej nazwy ("user agent string") słowa "Gecko", a nie spodziewać się nazwy konkretnego producenta przeglądarki lub zgodności całego "user string" z oczekiwaniami. Więcej szczegółów na ten temat zawiera artykuł:
http://developer.mozilla.org/pl/docs/Wykrywanie_wersji_i_programowanie_dla_wielu_przegl%C4%85darek
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages#Developing_cross_browser.2Fcross_platform_pages

http://developer.mozilla.org/pl/docs/Venkman
