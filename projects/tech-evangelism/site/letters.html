This letter is intended to be used as the basis for technical letters sent to web masters and web developers whose site's have issues with Mozilla. It should be customized to contain references to the specific URLs which exhibit problems, specific descriptions of the reasons why the content does not support Mozilla or other Gecko based browsers and if possible recommended changes to their content.

  1. Select All of the text in the letter and paste it into your email client,
  2. Edit the CUSTOMIZE sections to reflect the specific issues found at the site.
