POZOR: Vpis technicky popis problemu, nejlepe s moznymi spusobmi napravy. Muzes uvest priklady zdrojoveho kodu, jak by mela byt stranka zhotovena v soulade s webovymi standardy.

Podpora standardů W3C je potřebná, protože zajišťuje dostupnost Internetu uživatelům všech prohlížečů na všech platformách.

Podporou standardů W3C zpřístupníte obsah svého webu:

České překlady dokumentů W3C:

Ověření správnosti HTML a CSS
------------------------------

Na stránkách W3C se na adrese http://validator.w3.org/ nachází HTML Validator, který Vám pomůže ověřit správnost HTML kódu tak, aby se správně zobrazoval ve všech prohlížečích, které podporují standardy. W3C též připravilo soubor testů pro CSS 2.1 na adrese http://www.w3c.org/Style/CSS/Test/ a CSS 2.1 validator na adrese http://jigsaw.w3.org/css-validator/ Přihlédněte také k tomu, že v Mozille je výklad CSS velmi přísný a nesprávné CSS, které akceptují jiné prohlížeče, jako je MS Internet Explorer 7, mohou být odmítnuty. Prohlížeče založené na Mozille rozumějí každému CSS, který prošel testem na ověření jeho správnosti.

Články
------

Existuje množství článků (v angličtině), které Vám mohou pomoci při podpoře standardů:

Autorům webových stránek, jako jste Vy, je na mozilla.org věnovaná speciální sekce. Prosím, podívejte se na
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages
http://developer.mozilla.org/cs/docs/V%C3%BDvoj_pro_web

Následující článek se věnuje tvorbě WWW stránek tak, aby fungovaly ve všech prohlížečích:
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages:Developing_Cross_Browser/Cross_Platform_Pages

API jazyka DHTML je hlavní částí otevřených zdrojových souborů, které zabezpečují funkce na nastavení vlastností CSS Positioning nezávisle na prohlížeči. Rozhraní bylo aktualizováno tak, aby obsahovalo podporu nových standardů v prohlížečích založených na Mozille. Můžete ho najít na nové adrese: http://developer.mozilla.org/cs/docs/Webov%C3%A9_standardy
http://developer.mozilla.org/cs/docs/DOM

Na této adrese se můžete naučit, jak dynamicky měnit CSS vlastnosti s pomocí Javascriptu nezávislého na prohlížeči:
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages:Using_the_W3C_DOM#Accessing_Elements_with_the_W3C_DOM

Zde se nachází také Code Generator sloužící k vytváření Javascriptu na nastavení CSS vlastností, který nezávisí na prohlížeči a funguje v MS Internet Exploreru prohlížečích založených na Mozille.

Také firma Netscape má stránky věnované podpoře vytváření stránek nezávislých na prohlížeči pro Firefox 2 a ostatní prohlížeče založené na Mozille i MS Internet Exploreru 7 na adrese
http://developer.mozilla.org/cs/docs/Hlavn%C3%AD_strana

Prohlédněte si, prosím, "Practical Browser Sniffer" (praktický zjisťovač prohlížečů) na http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support

Naučte se, jak upravit DHTML stránky tak, aby podporovaly Firefoxu a Seamonkey na
http://developer.mozilla.org/en/docs/Gecko_DOM_Reference

Zjistěte, jak diagnostikovat, zda WWW stránky podporují Firefoxu a Seamonkey, na http://forum.czilla.cz/ (fórum Prohlížeč Firefox)
http://www.czilla.cz/
