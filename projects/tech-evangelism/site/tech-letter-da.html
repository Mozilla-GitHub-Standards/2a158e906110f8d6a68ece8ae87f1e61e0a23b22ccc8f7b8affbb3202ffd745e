Til webmaster,

Jeres hjemmeside TILPAS: indsæt URL her understøtter ikke Firefox 2 browser eller andre browsere baseret på layoutmotoren Gecko, som også benyttes af Camino 1, Seamonkey 1, Epiphany 2, Galeon 2 og K-meleon 1 .

Der er indsendt en rapport vedrørende dette problem i mozilla.org's fejlrapporteringssytem. Du kan se kommentarer og tilføje egne kommentarer angående problemet på følgende link:
https://bugzilla.mozilla.org/show_bug.cgi?id=TILPAS:indsæt fejlnummer her

TILPAS: Indsæt teknisk beskrivelse af problemerne her, sammen med råd og anbefalinger til hvordan problemet løses. Du kan også medsende fungerende eksempler på, hvordan sidens kode kan tilrettes så den overholder standarderne og kan ses i Mozilla.

Ved at følge ECMA-anbefalingerne til ECMAScript (JavaScript) og W3-anbefalingerne for HTML 4.01, CSS 2.1 og DOM (Document Object Model) i koden, vil jeres side virke i disse næste-generations browsere såvel som i andre browsere som understøtter åbne standarder som f.eks. Opera 9, Safari 2 og Internet Explorer 7.

Nedenstående ressourcer vil være en hjælp til at understøtte åbne standarder, samtidig med at jeres side virker med Microsofts Internet Explorer. Informationen omkring identifikation af Gecko-browseren er også vigtig.

ECMAScript 262 3rd Edition (JavaScript) Standard
---------------------------------------------------
http://www.ecma-international.org/publications/standards/Ecma-262.htm

Tekniske anbefalinger fra World Wide Web Consortium (W3C)
------------------------------------------------------------
W3C udarbejder de offentlige standarder for World Wide Web, som Firefox bruger.
http://www.w3.org/TR/
http://www.w3.org/2005/11/Translations/Lists/ListLang-da.html (Danske oversættelser af W3C dokumenter)

Mozilla Developer Center webudvikler dokumentation
------------------------------------------------
Besøg mozilla.orgs hjemmeside for webudviklere for at finde information om hvordan man specifikt understøtter Firefox 2 og andre Gecko-baserede browsere.
http://developer.mozilla.org/en/docs/Main_Page

Brug af webstandarder på jeres sider
---------------------------------------
En række elementer og vaner som blev opfattet som "standard" iblandt webudviklere for at tilføje DHTML og/eller "cross browser"-support til deres hjemmesider blev udelukket fra W3C HTML 4.01 og Document Object Model-specifikationerne. Find ud af hvad der understøttes på dette link.
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages

Mozilla's Document Object Model
----------------------------------
http://www.mozilla.org/docs/dom/ indeholder nyttige links for mere information om og brugen af Document Object Model (DOM) i Mozilla og andre Gecko-baserede browsere. Geckos DOM reference http://developer.mozilla.org/en/docs/Gecko_DOM_Reference indeholder en begynder-guide til implementationen af W3C DOM i Firefox og indeholder en række forskellige eksempler.

Opdarering af DHTML hjemmesider
----------------------------------
Alt hvad du behøver vide om document.all og andre properitære udvidelser og hvordan man arbejder med dem i en "cross browser-verden".
http://developer.mozilla.org/en/docs/Web_Standards

Browser dektektion og "cross browser" hjælp
----------------------------------------------
Når du kontrollerer om webstedets besøgende bruger Gecko-baserede browsere, er det nemmeste at søge efter "Gecko" i user-agent-strengen, ikke efter leverandørnavn eller en hel user-agent-streng. Du finder flere detaljer i denne artikel:
http://developer.mozilla.org/en/docs/Browser_Detection_and_Cross_Browser_Support
http://developer.mozilla.org/en/docs/Using_Web_Standards_in_your_Web_Pages#Developing_cross_browser.2Fcross_platform_pages

JavaScript-debugger til Firefox, Seamonkey
----------------------------------------------
Venkman er et komplet interaktivt javascript-fejlsøgsningsprogram til Firefox og Seamonkey som kører på enhver platform hvor Firefox og Seamonkey er understøttet. Læs venligst følgende for en introduktion:
http://developer.mozilla.org/en/docs/Venkman
http://www.svendtofte.com/code/learning_venkman/
