Carta Técnica para el Diseñador Web
  • Carta Técnica para el Diseñador Web (Castellano)

Esta carta está diseñada para ser usada como base para las cartas técnicas enviadas a los webmasters y diseñadores web cuyos sitios tengan problemas con Mozilla. Debe ser personalizada con referencias de los URL's específicos que presenten problemas, descripciones específicas de las razones por las cuales el contenido no soporta Mozilla u otros navegadores basados en Gecko y si es posible cambios recomendados al contenido.

  1. Selecciona todo el texto de la carta y pegalo en tu cliente de correo electrónico,,
  2. Edita las secciones personalizables para mostrar los problemas específicos encontrados en el sitio.
