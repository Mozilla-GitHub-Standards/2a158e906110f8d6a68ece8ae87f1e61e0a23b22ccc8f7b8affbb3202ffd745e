How to use the localized build and language packs
 
  • Localized builds (in compressed archives):
    1. Follow the link to download the .zip (or .gz for linux) file.
    2. Unzip it to your local drive into an empty folder, e.g., c:\mozilla.
    3. Remove the older Netscape or Mozilla profile data (see the release notes for your release)
    4. Run mozilla.exe (apprunner.exe in older releases).

  • Language pack distributed as compressed archives (M16 and earlier builds):
    1. Language packs are platform independent with some exceptions. It is sufficient to use the lang pack for the OS shown.
    2. You need to have a complete build installed before using language pack.
    3. Exit current Seamonkey client session.
    4. Follow the link to download the language pack in .zip (or .gz for linux).
    5. Unzip it into a temporary directory, e.g., c:\unzipped, on your local drive. You shall see a chrome\ subdirectory under c:\unzipped.
    6. Locate where your install the complete build. Assume it is in C:\Program Files\Netscape\Seamonkey
    7. Copy (or drag an drop) the whole chrome\ directory (or folder) into C:\Program Files\Netscape\Seamonkey
    8. Done!

  • Language packs installer, distributed as *.xpi files (available for M15-nb1b and later builds):
    1. You need to have a complete build installed before using a language pack.
    2. Language packs are platform independent with some exceptions. It is sufficient to use the lang pack for the OS shown.
    3. Click on the link under the "Installable Langpack" column.
    4. Follow the prompt. The installer script will download the language pack and install it automatically for you. Then restart Mozilla.
    5. For version later than 0.9.2 choose the installed language in Preferences | Appearance ; for prior versions use the menu item under View | Languages and Web Content. Restart Mozilla again.

 

Contact the original contributors or MLP Staff if you have problems using these language packs.
