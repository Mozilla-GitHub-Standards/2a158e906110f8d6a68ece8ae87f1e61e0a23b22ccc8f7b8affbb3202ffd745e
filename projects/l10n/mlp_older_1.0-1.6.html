How to use the localized build and language packs
 
  • Localized builds (in compressed archives):
    1. Follow the link to download the .zip (or .gz for linux) file.
    2. Unzip it to your local drive into an empty folder, e.g., c:\mozilla.
    3. Remove the older Netscape or Mozilla profile data (see the release notes for your release)
    4. Run mozilla.exe.

  • Language packs installer, distributed as *.xpi files:
    1. You need to have a complete build installed before using a language pack. You also need Javascript and online installation enabled (Chech your Mozilla preferences, from Edit | Preferences).
    2. Use the lang pack for the OS shown.
    3. Click on the link under the "Installable Langpack" column.
    4. Follow the prompt. The installer script will download the language pack and install it automatically for you.
    5. Choose the installed language in Preferences | Appearance.

 

Contact the original contributors or MLP Staff if you have problems using these language packs.
