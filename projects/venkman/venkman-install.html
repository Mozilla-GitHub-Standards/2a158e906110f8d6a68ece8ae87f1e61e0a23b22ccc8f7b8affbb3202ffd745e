Install Venkman 0.6.2...

The XPIs provided on this page can only be installed on browsers based on the Mozilla 0.9.4 codebase, including the Netscape 6.2x series. To set up Venkman on your browser, click on the appropriate ``install' link below and restart your browser. A "JavaScript Debugger" entry should show up in the Tasks->Tools menu.

Please see the Venkman Walkthrough for information on how to use this version of the Venkman.

These will NOT work on the latest Mozilla nightly builds. They will NOT work on Netscape 7.0 either. If you're not running a Mozilla 0.9.4 based browser (such as Netscape 6.2), you will most likely lose the ability to start your browser.

The Venkman Development Page has links to XPIs that will work on the latest Mozilla and Netscape releases.

If you ignore the above warning and your browser stops working, you can try the following steps...

  • Delete your mozilla directory and reinstall. This is the easiest way to deal with the problem.
  • If you don't want to do that then...
    • Quit Mozilla.
    • Delete the component.reg file.
    • Delete jsdservice.xpt and jsd3250.dll (windows), libjsd.so (linux) or jsdService.shlb (mac.)
    • Run regxpcom.

If your platform isn't listed here, you can help out by doing an optimized build of the Mozilla 0.9.4 source (gzip 34.3M, bzip2 26.4M) for your platform (make sure to build Venkman as described on the Venkman Homepage.) Then, send me the jsd binary from the components directory. On Unixish systems, this file will be called libjsd.so. I'll make an .xpi out of each binary I receive and post it here.
