Table Generator for URL-specific Field-to-Schema Mapping

This utility allows you to automatically create a mapping for forms at a URL of your choice and to have that mapping transmitted to Netscape for inclusion in the master table that is being maintained on their server.  To use the utility, perform the following steps:

1. Enter a URL in the textbox in the frame above. This need not be the URL of the page containing the desired form but could instead be the URL of a page from which you can navigate to get to that form.

2. Press the "Load URL" button.  The page at the URL you just entered will appear in this main frame in place of these instructions.

3. Navigate to the desired form if necessary.

4. Press the Update button.  This makes a copy of all the text fields and select lists in a frame at the right side.

5. In each text field, enter the corresponding schema name.  In other words, if the form contains text fields such as:

Name (first, middle, last):
you would enter the following:
Name (first, middle, last):
A list of the schema names to be used can be found here.

Each entry can be made either in the main frame (providing it fits) or in the side frame(it will always fit there).

6. For each select list, enter the schema name in the side frame.

7. Press the "Transmit Data" button.  This will transmit your values to Netscape for inclusion in Netscape's master table.

8. Hints:

a. Before transmitting the values (step 7), the table-generator performs a check to determine if the schema names you entered are valid.  Any error will be reported to you.  If there are errors, you can abort the transmission (by clicking Cancel when you receive the dialog "This form is being transmitted via e-mail").  At that point you can correct the erroneous schema names and then repeat step 7.

b. Some sites have forms that take over the entire display, obliterating the top and side frames used by this table generator.  In that case, copy the URL of the form itself, return to the table generator, and paste the URL into the top frame of the table-generator display

c. Some sites have their forms inside their own frames, preventing the Update button (step 4) from finding the information.  In that case, use the browser's "View Page Info" command to determine the URL of the particular frame of interest.  Then copy that URL and paste it directly into the top frame of the table-generator display.
