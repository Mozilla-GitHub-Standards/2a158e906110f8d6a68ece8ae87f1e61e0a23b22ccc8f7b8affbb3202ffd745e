Turbo Mode (QuickLaunch) Plan


Functionality Spec

When the user has enabled QuickLaunch either through the installer or via preferences (preferences->advanced),, an icon will appear in the windows system tray signalling that turbo mode (QuickLaunch) is active, either immediately if invoked via preferences or on next launch if turned on via the installer (which launches mozilla anyway). When QuickLaunch is active, double clicking the mozilla.exe or the icon in the system tray will cause a new window to be created using the already running mozilla process.

The system tray icon also presents a menu when the user right-clicks on it. The menu items are:
  • Navigator
  • Main & Newsgroups
  • Composer
  • Address Book
  • (-- menu seperator --)
  • Disable QuickLaunch
  • Exit Mozilla
Selecting any of the mozilla components opens that component. Disable QuickLaunch disables the QuickLaunch preference and shuts down turbo mode, but will leave open any mozilla windows. The "Exit Mozilla" is disabled if there are any open mozilla windows. Selecting exit will really quit mozilla (but not alter the QuickLaunch preference) such that relaunching mozilla causes a full relaunch (hopefull a warm relaunch ;-) ).
