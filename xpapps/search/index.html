xpapps::search



The search feature has multiple entry points in mozilla including the URL bar autocomlpetion dropdown, the URL bar search button, the context menu when text is selected, the search sidebar tab, and the search menu items.  The feature exposes "basic" searching for newcomers to the web as well as "advanced" searching for more experienced web surfers.  Search is already quite configurable: basic/advanced mode selection, default search engine selection, and more.  The backend is based on the Sherlock format as specified by Apple.  We have extended the format to support some mozilla-specific features inclduing constraining the basic list.

This page shall serve as the launching pad for search development in mozilla.  Following are links to relevant working documents and proposals.
  • Search and Navigation UI Spec
  • Mach V Engineering Plan
