<treehead>
Attributs Enfants communs
id treerow
  treecol
  treeitem

 
L'élément <treehead> est un conteneur spécial pour les lignes d'arbre qui apparaissent en haut du widget <tree>. <treehead> est un enfant de l'élément <tree>. Les noeuds enfants de <treehead> apparaissent en haut de la définition de l'arbre, tout comme le sous-arbre <thead> dans une table HTML.

Pour plus d'information sur la structure du widget <tree>, voir la description tree.
