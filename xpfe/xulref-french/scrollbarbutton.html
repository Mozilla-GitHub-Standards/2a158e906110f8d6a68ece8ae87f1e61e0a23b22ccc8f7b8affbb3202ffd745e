Le scrollbarbutton est le widget bouton à l'intérieur du scrollbar. Il hérite du titledbutton et a certains comportements supplémentaires. La classe du bouton détermine quel genre de bouton c'est. L'incrément est une classe pour XXX. L'exemple suivant montre un scrollbarbutton défini dans un sous-arbre scrollbar:

 
<scrollbar align="vertical" 
    curpos="50" 
    pageIncrement="20" 
    increment="5" 
flex="100%">
  <scrollbarbutton type="decrement"/>
  <slider flex="100%">
     <thumb value="Thumb" flex="0%"/>
  </slider>
  <scrollbarbutton type="increment"/>
</scrollbar>

 

 
type
Description
L'attribut type indique l'orientation du scrollbarbutton sur le scrollbar.
Syntaxe 
<scrollbarbutton type="decrement | increment" />
Exemple
<scrollbarbutton type="decrement" />
Notes
Les scrollbarbuttons "Decrement" ont des flèches dirigées vers le haut parce qu'elles font défiler la page vers le haut. Les types "Increment" ont des flèches descendantes. L'exemple ci-dessus montre un scrollbarbutton de décroissance, puis un glisseur (slider) avec un enfant, puis un scrollbarbutton d'incrément.
