value
Description
value est un attribut exigé employé pour fournir une étiquette pour le bouton.
Syntaxe
<titledbutton value="string" />
Exemple
<titledbutton id="pushme" class="push" value="Push Me" />
Notes
Il est légal de ne pas fournir de valeur pour un bouton, mais le bouton peut être invisible s'il n'y a pas valeur pour contraindre la taille réelle du bouton ni aucune information concrète qui indique ses positions et tailles.

 

 
onclick
Description
onclick est un attribut auditeur d'événement utilisé pour ajouter une fonctionnalité aux boutons XUL. 
Syntaxe
<titledbutton onclick="emplacement javascript ou appel à une fonction javascript"
Exemple
<titledbutton value="Print" id="print" onclick="alert('Printing Document'); Print();" />
Notes
L'auditeur d'événement onclick est une méthode d'appel des commandes des boutons. Notez que oncommand est la manière préférée d'attacher un dérouleur d'événement à un bouton. onclick est désapprouvé, bien qu'encore disponible.

L'événement de clique est actionné toutes les fois que l'utilisateur clique le bouton. Vous pouvez mettre du JavaScript en ligne avec l'auditeur d'événement, vous pouvez appeler des fonctions JavaScript définies dans la section <head> du fichier XUL dans un élément <html:script>...</html:script>, ou vous pouvez appeler des fonctions JavaScript définies dans un fichier source séparé qui est chargé avec la construction <html:script language="javascript" src="source_file_name.js" />

 
 
 
oncommand
Description
oncommand est un attribut auditeur d'événement utilisé pour ajouter la fonctionnalité aux boutons XUL. 
Syntaxe
<titledbutton oncommand="emplacement javascript ou appel à une fonction javascript"
Exemple
<titledbutton value="Print" id="print" oncommand="alert('Printing Document'); Print();" />
Notes
L'auditeur d'événement oncommand est une manière commune d'appeler des commandes des boutons. Elle substitue onclick comme voie principale pour attacher un dérouleur d'événement à un titlebutton.

Vous pouvez mettre du JavaScript en ligne avec l'auditeur d'événement, vous pouvez appeler des fonctions JavaScript définies dans la section <head> du fichier XUL dans un élément <html:script>...</html:script>, ou vous pouvez appeler des fonctions JavaScript définies dans un fichier source séparé qui est chargé avec la construction <html:script language="javascript" src="source_file_name.js" />


 
tooltiptext
Description
tooltiptext est un est un attribut facultatif employé pour indiquer le texte tooltip pour le bouton.
Syntaxe
<titledbutton tooltiptext="string" />
Exemple
<titledbutton id="send_mail" class="push" tooltiptext="Send Mail (Ctrl + M)" />
Notes
Si vous voulez afficher du texte structuré ou plus long dans la fenêtre tooltip, vous devez utiliser l'attribut de tooltip à la place.

toggled
Description
toggled est un attribut facultatif employé pour indiquer l'état  (basculé) d'un bouton.
Syntaxe
<titledbutton toggled="0 | 1 | 2" />
Exemple
<titledbutton id="local" class="push" toggled="1" />
Notes
0 signifie qu'il n'est pas basculé, 1 signifie qu'il l'est et 2 signifie que les deux sont possibles.

Description
src est un attribut facultatif pour le chargement d'images du bouton.
Syntaxe
<button src="URL" />
Exemple
<button id="security" src="images\lock_and_key.gif" />
Notes
Aucune.

 

 
crop
Description
crop est un attribut facultatif pour spécifier la partie texte d'un bouton. Le cas échéant, il sera coupé s'il est trop grand.
Syntaxe
<button id="string" value="Nom de bouton" crop="left | right | none | center" >
Exemple
<button id="string" value="Très long nom de bouton" crop="none" >
Notes
Aucune.
