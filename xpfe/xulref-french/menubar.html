<menubar> est le principal widget dans la définition d'un menubar XUL. Les browsers et autres applications XUL peuvent contenir un nombre indéfini de menubars. Typiquement, un menubar contient un certain nombre de menus (par exemple, File, Edit, View, etc.), dont chacun définit un menu de commandes disponibles à l'utilisateur. Mais les menubars peuvent avoir d'autres éléments comme enfants. Comme les toolbars, les menubars peuvent contenir presque tous les éléments XUL comme enfants. Le menubar hérite de l'élément de cadre de base (box).
