L'élément <menubutton> est un élément du menu qui inclut un graphisme ou une image. Excepté src, tous ses attributs sont dérivés de <menu>

 

 

src
Description
src est un attribut facultatif utilisé pour indiquer le graphisme de menu de menubutton.
Syntaxe
<menubutton src="image file">
Exemple 
<menubutton src="back.gif">
<menupopup>
  <menuitem value="foo" />
  <menuitem value="shoo" />
  <menuitem value="boo" />
</menupopup>
</menulist>
