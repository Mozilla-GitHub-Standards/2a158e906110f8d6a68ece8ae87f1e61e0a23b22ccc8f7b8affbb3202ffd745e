<treechildren>
Attributs Enfants communs
id treeitem
open  
L'élément <treechildren> est un des principaux conteneur dans un widget tree, en se plaçant typiquement juste sous l'élément principal <tree>. Treechildren est parent des éléments qui apparaissent réellement dans l'arbre (par opposition à <treehead>, qui contient normalement les éléments non-visibles).
 

 
open

Description

L'attribut open indique si un élément treechildren donné est élargi ou rendu compact.

Syntaxe

<treechildren open="true | false" />

Exemple

<tree>
  <treeheader></treeheader>
  <treechildren id="Main" open="true" />
     <treeitem ...
  </treechildren>
</tree>

Notes

Vous pouvez faire  persister l'attribut open d'un élément treechildren afin de préserver l'état du treechildren quand l'application est fermée.
