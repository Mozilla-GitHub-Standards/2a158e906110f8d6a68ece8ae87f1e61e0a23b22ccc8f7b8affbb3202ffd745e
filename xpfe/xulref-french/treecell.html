<treecell>
Attributs Enfants communs
id * * *
align

<treecell> est l'unité atomique d'une structure arbre; il est équivalent à l'élément <td> d'une table HTML.
 

 
crop

Description

crop est un attribut facultatif pour spécifier quelle partie du texte d'une cellule d'arbre sera, le cas échéant, coupée s'il est trop grand.

Syntaxe

<treecell crop="left | right | none | center" />

Exemple

<treecell value="list_item2" crop="none" />

Notes

La valeur par défaut pour l'attribut crop est "none".

 
indent

Description

indent est un attribut facultatif qui spécifie si une cellule d'arbre est indentée ou non.

Syntaxe

<treeitem indent="true | false" />

Exemple

<treecell value="list_item2" indent="false" />

Notes

La valeur par défaut pour l'attribut container est "false".

 
value

Description

value est un attribut facultatif qui fournit l'étiquette des textes pour la cellule.

Syntaxe

<treecell value="text label" >

Exemple

<treecell value="Favorite Web Sites" >

Notes

Les attributs value de beaucoup d'éléments sont exigés. Mais à la différence de quelques éléments, <treecell> a un attribut value et un contenu (cad, tag <treecell></treecell> entourant un certain contenu). Quand aucune valeur n'est indiquée pour value, le contenu des cellules de l'arbre est utilisé directement. Par exemple, l'élément suivant apparaît dans un arbre en tant que "Helpful Links":

<treecell id="helpful">Helpful Links</treecell>
