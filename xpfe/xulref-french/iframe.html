<iframe> est un widget fonctionnel et encastrable qui fournit la fonctionnalité d'édition à l'utilisateur. Employez l'attribut src pour charger le contenu dans la fenêtre iframe (y compris src="about:blank") pour indiquer une nouvelle page blanche pour l'édition. Et voir les types indiqués dans NS6 pour se familiariser avec le bla bla...
 

 
type
Description
type est un attribut facultatif qui indique le type de la fenêtre de contenu.
Syntaxe 
<iframe type="type" ...>
Exemple 
<iframe type="content-primary" id="main-window">
Notes
"content-primary" est le type utilisé pour l'iframe principal dans Netscape 6. Comme l'attribut class affecte la manière dont le chrome est rendu, l'attribut type affecte la manière dont le contenu est traité dans le browser. Les différents types d'iframe apparaissant dans le browser de Mozilla incluent:
  • content-primary dénote la fenêtre iframe principale
  • content est un autre type commun pour les iframes comme le Composer dans l'application Mozilla

 


 
name
Description
name est un attribut facultatif qui indique le nom de la fenêtre de contenu.
Syntaxe 
<iframe name="name" ...>
Exemple 
<iframe name="content-primary" id="main-window">
Notes
L'attribut name est utilisé au lieu de id pour mettre en référence une fenêtre de contenu à parties d'autres parties du chrome, comme quand un bouton charge le nouveau contenu en changeant l'attribut <src> sur l'iframe appelée:

  
<iframe name="main-iframe">

...

function loadPage() {
  ed = document.getElementById("main-iframe")
  ed.setAttribute("src", "newpagetoedit.html")
}

Vous ne pouvez pas chercher la fenêtre de contenu (cad. getElementById(id)) en utilisant l'attribut id.


 


 
src
Description
src est l'attribut employé pour charger le contenu dans la fenêtre de contenu.
Syntaxe 
<iframe src="fichier de contenu HTML" ...>
Exemple 
<iframe id="main-window" src="about:blank">
Notes
Utilisez JavaScript pour mettre l'attribut src à partir de l'IU:

  
<iframe name="main-iframe">

...

function loadPage() {
  ed = document.getElementById("main-iframe")
  ed.setAttribute("src", "newpagetoedit.html")
}


 


 
onblur
Description
onblur est un attribut auditeur d'événement pour manipuler l'événement actuel quand l'iframe détruit le focus.
Syntaxe
<element onblur="code javascript dérouleur d'évènement">
Exemple 
<iframe onblur="blurWindow()">
Notes
Aucune.


 
onfocus
Description
onfocus est un attribut auditeur d'événement pour manipuler l'événement actuel quand l'iframe obtient le focus de l'utilisateur.
Syntaxe
<element onfocus="code javascript dérouleur d'évènement">
Exemple 
<iframe onblur="focusWindow()">
Notes
Aucune.
