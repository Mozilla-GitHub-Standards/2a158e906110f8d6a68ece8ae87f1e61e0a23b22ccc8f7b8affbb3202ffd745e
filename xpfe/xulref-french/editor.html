<editor> est un widget fonctionnel et encastrable qui permet d'éditer une fonctionnalité à l'utilisateur. Employez l'attribut src pour charger le contenu dans la fenêtre d'édition (y compris src="about:blank") afin d'indiquer une nouvelle page blanche pour l'édition. Voir les types indiqués dans NS6 pour se familiariser avec le bla bla..
 

 
type
Description
type est un attribut facultatif qui indique le type de la fenêtre de contenu.
Syntaxe 
<editor type="type" ...>
Exemple 
<editor type="content-primary" id="main-window">
Notes
"content-primary" est le type utilisé pour l'éditeur principal dans Netscape 6. Juste comme l'attribut class affecte la manière dont le chrome est rendu, l'attribut de type affecte la manière dont le contenu est traité dans le browser. Les différents types d'éditeur apparaissant dans le browser de Mozilla incluent:
  • content-primary dénote la fenêtre principale d'édition
  • content est un autre type commun pour les éditeurs Composer-like dans l'application Mozilla

 


 
name
Description
name est un attribut facultatif qui indique le nom de la fenêtre de contenu.
Syntaxe 
<editor name="name" ...>
Exemple 
<editor name="content-primary" id="main-window">
Notes
L'attribut name est utilisé au lieu de id pour mettre en référence une fenêtre de contenu d'autres parties du chrome, comme quand un bouton charge un nouveau contenu en changeant l'attribut <src> sur l'éditeur nommé:

  
<editor name="main-editor">

...

function loadPage() {
  ed = document.getElementById("main-editor")
  ed.setAttribute("src", "newpagetoedit.html")
}

Vous ne pouvez pas chercher le contenu de la fenêtre (cad. getElementById(id)) en utilisant l'attribut id.


 


 
src
Description
src est l'attribut employé pour charger le contenu dans la fenêtre de contenu.
Syntaxe 
<editor src="fichier de contenu HTML" ...>
Exemple 
<editor id="main-window" src="about:blank">
Notes
Utiliser JavaScript pour placer l'attribut src de l'IU:

  
<editor name="main-editor">

...

function loadPage() {
  ed = document.getElementById("main-editor")
  ed.setAttribute("src", "newpagetoedit.html")
}


 


 
onblur
Description
onblur est un attribut auditeur d'événement pour manipuler l'événement actuel quand l'éditeur détruit le focus.
Syntaxe
<element onblur="code javascript dérouleur d'évènement">
Exemple 
<editor onblur="blurWindow()">
Notes
Aucune.


 
onfocus
Description
onfocus est un attribut auditeur d'événement pour manipuler l'événement actuel quand l'éditeur obtient le focus de l'utilisateur.
Syntaxe
<element onfocus="code javascript dérouleur d'évènement">
Exemple 
<editor onblur="focusWindow()">
Notes
Aucune.
