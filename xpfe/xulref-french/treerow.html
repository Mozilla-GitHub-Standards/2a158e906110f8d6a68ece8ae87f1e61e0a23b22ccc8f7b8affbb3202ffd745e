Manuel de référence du programmeur XUL

<treerow>
Attributs Enfants communs
id treecell
   

 
<treerow> est la ligne réelle dans la structure d'arbre. Il est comparable à l'élément <td> dans des tables HTML. Les étiquettes suivantes sont analogues à leur équivalent dans une table HTML.
 
Table Tree
table tree
caption treecaption
colgroup treecolgroup
col treecol
thead treehead
tbody treechildren, treeitem
tfoot treefoot
tr treerow
td treecell
Les éléments treehead et treechildren ont tous les deux treerow en tant qu'enfant, qui définit alternativement les éléments <treecell> comme enfants.
 

Voir <tree> pour plus d'information sur les structures arborescentes.
