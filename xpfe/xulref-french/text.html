<text />

Attributs Enfants communs

<text> est un widget tout usage d'affichage de texte. Il peut être utilisé dans les endroits où vous voulez afficher le texte lui-même, comme vous pourriez avoir fait précédemment avec une étiquette HTML ou comme du texte faisant partie d'un widget XBL composite.

 

value
Description
L'attribut value indique le texte affiché pour l'élément text.
Syntaxe
<text value="string" />
Exemple 
<text value="Pour plus d'information, cliquer sur Aide" />
Notes
Aucune.
 
class
Description
L'attribut class indique une classe particulière de widget text pour une utilisation d'affichage.
Syntaxe
<text class="string" />
Exemple 
<checkboxvalue="Honda"/>
<checkbox value="Isuzu" />
Notes
Une classe simple dans le skin global, text class="label", ajoute un remplissage supplémentaire autour du texte. Hormis celui-ci, il n'y a aucune classe spécialisée pour habiller ce widget XUL à usage universel.
 
for
Description
L'attribut for indique que l'élément text doit être utilisé uniquement en tant qu'élément XXX.
Syntaxe
<text for="id de l'élément cible" />
Exemple 
<textfor="button1" value="Voir Source" />
<button for="button1" />
Notes
