Un élément <radio> apparaît dans un radiogroup. Dans un radiogroup, un seul élément radio peut être choisi en même temps.

  
<?xml-stylesheet href="chrome://global/skin/" type="text/css"?> 

<window id="radio show" 
  xmlns:html="http://www.w3.org/1999/xhtml"
  xmlns="http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul">

  <box style="background-color: lightblue;">
	
    <radiogroup orient="vertical">
	<radio value="James Joyce" />
	<radio value="Joseph Conrad" />
	<radio value="William Faulkner" />
    </radiogroup>
  </box>
</window>
 
value
Description
L'attribut value indique le nom affiché pour l'élément radio.
Syntaxe
<radio value="string" />
Exemple 
<radio value="Honda"/>
<radio value="Isuzu" />
Notes
Aucune.
checked
Description
Quand checked est mis à "true", l'élément radio est pré-sélectionné dans le radiogroup.
 
Syntaxe
checked="[true | false]"
Exemple 
<?xml-stylesheet href="chrome://global/skin/" type="text/css"?> 

<window id="radio show" 
  xmlns:html="http://www.w3.org/1999/xhtml"
  xmlns="http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul">

  <box style="background-color: lightblue;">
	
    <radiogroup orient="vertical">
	<radio value="James Joyce" />
	<radio value="Joseph Conrad" checked="true" />
	<radio value="William Faulkner" />
    </radiogroup>
  </box>
</window>
Notes
Aucune.
 
oncommand
Description
oncommand est un attribut facultatif auditeur d'événement pour l'élément radio.
Syntaxe
<radio id="string" value="string" oncommand="code dérouleur d'évènement" />
Exemple
<radio id="saveKey" value="Honda" oncommand="doHondaPrep()" />
Notes
Aucune.

 

 
crop
Description
crop est un attribut facultatif qui fait partie du texte de l'élément radio. Le cas échéant, il est coupé quand le conteneur parent ne fournit pas assez d'espace.
Syntaxe 
<radio crop="left | right | none" />
Exemple 
<radio value="Long nom pour cet élément Radio" crop="left" />
Notes
Aucune.

 

 
imgalign
Description
imgalign est un attribut facultatif pour aligner le graphique importé avec l'élément radio.
Syntaxe 
<radio src="graphic" 
imgalign="
  absbottom |
  absmiddle |
  baseline  |
  bottom    |
  left      |
  middle    |
  right     |
  texttop   |
  top" />
Exemple
<radio src="newbutton.gif" imgalign="absmiddle" />
Notes
Aucune.

 

 
src
Description
src est un attribut facultatif pour indiquer des images graphiques à utiliser au lieu du bouton radio lui-même.
Syntaxe
<radio src="chrome ou url régulière vers une image" />
Exemple
<radio src="chrome://global/newradio.gif" />
Notes
Aucune.
