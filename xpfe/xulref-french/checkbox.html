L'élément <checkbox> fournit un widget de bouton spécial que l'utilisateur peut valider et invalider. A la différence des radio buttons, les checkbox ne sont groupés entre elles d'aucune manière, ainsi leur validation et invalidation n'est pas liée ou mutuellement exclusive.

  
<?xml-stylesheet href="chrome://global/skin/" type="text/css"?> 

<window id="checkboxes" 
  xmlns:html="http://www.w3.org/1999/xhtml"
  xmlns="http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul">

  <box style="background-color: lightblue;" orient="vertical">
	<checkbox value="James Joyce" />
	<checkbox value="Joseph Conrad" />
	<checkbox value="William Faulkner" />
  </box>
</window>
 
value
Description
L'attribut value de valeur indique le nom affiché pour l'élément checkbox.
Syntaxe
<checkbox value="string" />
Exemple 
<checkboxvalue="Honda"/>
<checkbox value="Isuzu" />
Notes
Aucune.
checked
Description
Une fois que la validation (checked) est placée à "vrai", le checkbox est pré-sélectionné.
 
Syntaxe
checked="[true | false]"
Exemple 
<?xml-stylesheet href="chrome://global/skin/" type="text/css"?> 

<window id="checkboxes" 
  xmlns:html="http://www.w3.org/1999/xhtml"
  xmlns="http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul">

  <box style="background-color: lightblue;" orient="vertical">
	<checkbox value="James Joyce" />
	<checkbox value="Joseph Conrad" checked="true" />
	<checkbox value="William Faulkner" />
  </box>
</window>
Notes
Aucune.
 
oncommand
Description
oncommand est un attribut facultatif auditeur d'événement pour le checkbox.
Syntaxe
<checkbox id="string" value="string" oncommand="code auditeur d'évènement" />
Exemple
<checkbox id="whichCar" value="Honda" oncommand="doHondaPrep()" />
Notes
Aucune.

 

 
crop
Description
crop est un attribut facultatif qui fait partie du texte de checkbox. Le cas échéant, il est coupé quand le conteneur parent ne fournit pas assez d'espace.
Syntaxe 
<checkbox crop="left | right | none" />
Exemple 
<checkbox value="Nom long pour cette checkbox"
crop="left" />
Notes
Aucune.

 

 
imgalign
Description
imgalign est un attribut facultatif pour aligner le graphique importé avec checkbox.
Syntaxe 
<checkbox src="graphic" 
imgalign="
  absbottom |
  absmiddle |
  baseline  |
  bottom    |
  left      |
  middle    |
  right     |
  texttop   |
  top" />
Exemple
<checkbox src="newbutton.gif" imgalign="absmiddle" />
Notes
Aucune.

 

 
src
Description
src est un attribut facultatif pour indiquer des images graphiques à utiliser au lieu du checkbox par défaut.
Syntaxe
<checkbox src="chrome ou url régulière vers une image" />
Exemple
<checkbox src="chrome://global/newbox.gif" />
Notes
Aucune.
