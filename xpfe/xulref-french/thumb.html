Le thumb est le bouton mobile dans le scrollbar. Comme scrollbar, les attributs de thumb déterminent la position et la cadence de défilement de l'élément. L'exemple suivant montre un thumb défini dans un scrollbar:
 
<scrollbar align="vertical" 
    curpos="50" 
    pageIncrement="20" 
    increment="5" 
    flex="100%">
  <scrollbarbutton type="decrement"/>
  <slider flex="100%">
     <thumb value="Thumb" flex="0%"/>
  </slider>
  <scrollbarbutton type="increment"/>
</scrollbar>


 
curpos
Description
curpos est un attribut exigé pour indiquer la position du bouton du thumb quand il charge. Le nombre est un nombre de 0 à maxpos. 
Syntaxe
<thumb curpos="nombre de pixels descendus de 0 à maxpos" />


Exemple

<thumb curpos="10" />


Notes

Aucune.


 
maxpos
Description
maxpos est un attribut facultatif pour indiquer la longueur scrollable du scrollbar. 
Syntaxe
<thumb maxpos="nombre de pixels" />


Exemple

<scrollbar id="string" pageIncrement="XXX" />


Notes

maxpos et curpos s'associent entre eux pour déterminer la longueur scrollable et la position du thumb.


 
 
increment
Description
increment est un attribut facultatif pour indiquer de combien la position change sur chaque clique de scrollbarbutton et de la zone de slider. 
Syntaxe
<thumb increment="nombre de pixels à scroller" />


Exemple

<thumb increment="5" />


Notes

La valeur par défaut de increment est 1.

 
 
 
pageIncrement
Description
pageIncrement est un attribut facultatif pour indiquer le degré de changement de position quand l'utilisateur utilise le scrollbar.  
Syntaxe
<thumb pageIncrement="nombre de pixels à scroller sur la page" />


Exemple

<thumb pageIncrement="25" />


Notes

Utiliser "Page Up" et "Page Down" pour de plus grandes parties d'une zone scrollable selon la valeur indiquée pour pageIncrement. Voir également increment pour les degrés de défilement. La valeur par défaut de pageIncrement est 20.
