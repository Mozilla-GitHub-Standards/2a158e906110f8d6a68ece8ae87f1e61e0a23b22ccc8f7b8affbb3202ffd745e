<tabcontrol>

<tabcontrol> est l'élément supérieur dans le positionnement du widget tab. Comme l'élément <window> pour les fichiers XUL, <tabcontrol> doit encapsuler tous les autres éléments connexes à tab afin de travailler correctement. Tabcontrol déclare les widgets tab utilisés dans le fichier donné XUL. Comme plusieurs des autres widgets tab, le tabcontrol hérite du cadre box, si bien que tous les attributs de disposition associés à cet élément peuvent être utilisés ici. L'exemple suivant montre <tabcontrol> comme élément racine d'un widget tab de base:

  
 
<tabcontrol align="vertical"> 
  <tabbox align="horizontal"> 
        <tab value="Tab One" /> 
        <tab value="Tab Two" /> 
        <tab value="Tab Three" /> 
  </tabbox> 
                              
  <tabpanel flex="100%"> 
        <text value="View
One"/> 
        <text value="View
Two"/> 
        <text value="View
Three"/> 
  </tabpanel> 
</tabcontrol> 
Notez que l'aspect le plus souvent associé aux répertoires tabulés est réalisé en utilisant l'attribut align="vertical" sur tabcontrol. Le fait d'aligner les widgets tab horizontalement, bien qu'utile dans certains cas, place les panneaux tabulés près des tabulateurs eux-mêmes.
