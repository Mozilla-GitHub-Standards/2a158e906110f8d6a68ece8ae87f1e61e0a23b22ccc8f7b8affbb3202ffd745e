<browser> est un widget fonctionnel et encastrable qui permet d'éditer une fonctionnalité pour l'utilisateur. Employez l'attribut src pour charger le contenu dans la fenêtre de browser (incluant src="about:blank") afin d'indiquer une nouvelle page blanche pour l'édition. Et voir les types indiqués dans NS6 pour se familiariser avec le bla bla..
 

 
type
Description
type est un attribut facultatif qui indique le type du contenu de la fenêtre.
Syntaxe 
<browser type="type" ...>
Exemple 
<browser type="content-primary" id="main-window">
Notes
"content-primary" est le type utilisé par le browser principal dans Netscape 6. De la même manière que l'attribut class affecte la façon dont le chrome est rendu, l'attribut type affecte la manière dont le contenu est traité dans le browser. Les différents types de browser apparaissant dans le browser de Mozilla incluent:
  • content-primary dénote la fenêtre de browser principal
  • content est un autre type commun pour des browsers comme le Composer dans l'application Mozilla

 


 
name
Description
name est un attribut facultatif qui indique le nom de la fenêtre de contenu.
Syntax 
<browser name="name" ...>
Example 
<browser name="content-primary" id="main-window">
Notes
L'attribut name est utilisé au lieu de id pour mettre en référence une fenêtre de contenu pour d'autres parties du chrome, comme quand un bouton charge le nouveau contenu en changeant l'attribut <src> sur le browser:

  
<browser name="main-browser">

...

function loadPage() {
  ed = document.getElementById("main-browser")
  ed.setAttribute("src", "newpagetoedit.html")
}

Vous ne pouvez pas chercher la fenêtre de contenu (cad. getElementById(id)) en utilisant l'attribut id.


 


 
src
Description
src est l'attribut employé pour charger le contenu dans la fenêtre de contenu.
Syntax 
<browser src="fichier de contenu HTML" ...>
Example 
<browser id="main-window" src="about:blank">
Notes
Utilisez JavaScript pour affecter l'attribut src de l'IU:

  
<browser name="main-browser">

...

function loadPage() {
  ed = document.getElementById("main-browser")
  ed.setAttribute("src", "newpagetoedit.html")
}


 


 
onblur
Description
onblur est un attribut auditeur d'événement pour manipuler l'événement en cours quand le browser perd le focus.
Syntaxe
<element onblur="code javascript dérouleur d'évènement">
Exemple 
<browser onblur="blurWindow()">
Notes
Aucune.


 
onfocus
Description
onfocus est un attribut auditeur d'événement pour manipuler l'événement en cours quand le browser obtient le focus de l'utilisateur.
Syntaxe
<element onfocus="code javascript dérouleur d'évènement">
Exemple 
<browser onblur="focusWindow()">
Notes
Aucune.
