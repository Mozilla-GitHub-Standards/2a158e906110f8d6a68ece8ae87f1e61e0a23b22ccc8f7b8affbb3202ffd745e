<tab>  est l'actuel tabulateur dans le positionnement de tabulation. Le nom du tabulateur est pris à partir du contenu plutôt que d'un attribut et la tabulation se dimensionne automatiquement pour s'adapter au nom du tabulateur. Le widget <tab> n'hérite pas du cadre, mais il peut être indiqué comme n'importe quel widget de XUL. L'information de modèle pour le widget de tabulateur est contenue dans le fichier global.css de Mozilla.
  
 
<tabcontrol align="vertical">
   <tabbox align="horizontal">
         <tab value="Tab One" />

value

Description

L'atttribut value spécifie le texte du tabulateur.

Syntaxe

<tab value="string" />

Exemple

<tab value="Options" />

Notes

L'utilisation de l'élément <tab> comme conteneur pour le texte réel du tabulateur (cad, <tab>Options</tab>) n'est plus légale dans XUL. Le widget de tabulateur, comme la plupart des widgets dans le toolkit, hérite du cadre et des attributs d'utilisation comme value pour contrôler les étiquettes et le texte.
 
selected

Description

L'attribut selected indique si un treeitem donné est pré-sélectionné ou pas.

Syntaxe

<treeitem selected="true | false" />

Exemple

<tab value="index" selected="true" />

Notes

Vous pouvez obtenir et placer l'attribut selected en utilisant le DOM:
tb = document.getElementById("tab1")
tb.setAttribute("selected", "true")
