<treeitem>
Attributs Enfants communs
id treehead
container treechildren
open  

 
<treeitem> est un conteneur pour les éléments enfants dans la structure arborescente. Un treeitem se place entre les éléments réels (tels que les treerows ou les treecells, qui sont les éléments visibles dans l'arbre) et les treechildren ou les conteneurs treehead.
 

 
container

Description

L'attribut container indique si un treeitem donné est un conteneur ou pas.

Syntaxe

<treeitem container="true | false" />

Exemple

<treeitem id="tab-data" value="leaf" container="false" />

Notes

La valeur par défaut de l'attribut container est "false".
