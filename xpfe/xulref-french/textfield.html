L'élément textfield fournit un widget "text-entry" d'entrée de texte tout comme le widget HTML <input type="textfield" />. Vous pouvez employer l'attribut value pour mettre le texte dans le champ quand il est chargé et multiline pour utiliser le textfield comme une zone de texte multiligne.
 
 
value
Description
L'attribut value indique le texte affiché dans le champ de texte textfield.
Syntaxe
<textfield value="string" />
Exemple 
<textfield value="Entrer l'adresse du conteneur ici" />
Notes
Aucune.
 
multiline
Description
L'attribut multiline change le champ de texte 'une seule ligne' en widget textarea.
Syntaxe
<textfield multiline="true | false" />
Exemple 
<textfield value="Fill in here" />

<textfield 

  value="Je suis une zone textarea maintenant" 
  multiline="true"/>
Notes
Quand l'utilisateur saisit le texte multiligne, la zone défilera horizontalement si l'entrée est plus longue que la largeur disponible. Vous devez explicitement taper un retour-chariot pour écrire le texte sur la prochaine ligne.
