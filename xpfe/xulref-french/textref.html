<text> est un widget tout usage d'affichage de texte. Il peut être utilisé dans les endroits où vous voulez afficher le texte lui-même, comme vous pourriez avoir fait précédemment avec une étiquette HTML ou comme du texte faisant partie d'un widget XBL composite.

L'élément textfield fournit un widget "text-entry" d'entrée de texte tout comme le widget HTML <input type="textfield" />. Vous pouvez employer l'attribut value pour mettre le texte dans le champ quand il est chargé et multiline pour utiliser le textfield comme une zone de texte multiligne.

La différence entre les deux widgets peut être vue dans l'exemple suivant: 

<textfield value="Fill in here" />

<textfield 

  value="I am a textarea now" 
  multiline="true"/>
