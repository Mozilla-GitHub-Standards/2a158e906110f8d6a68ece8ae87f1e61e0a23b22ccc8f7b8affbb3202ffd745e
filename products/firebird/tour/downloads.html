Downloading Files

Mozilla Firebird automates file downloading and handling in a way never before seen in browsers on Microsoft Windows. By enabling Automatic Downloading, Firebird will automatically download files to your chosen location (e.g. your desktop) which removes the jarring hassle of dealing with the Save As dialog every time you download a file. This makes all file downloading, and especially file collection, much more efficient.



 

Firebird also now includes comprehensive Helper Application support, so you can easily associate files with handlers.

 



Firebird also includes a utility that lets you disable Plugin handling for links, so that (for example) when you click a link to a .mov file, you can choose to download that file to disk or launch it in the external Quicktime player, rather than being forced to view the file in the browser window in the plugin (which often has limited functionality). This also significantly eases media collection.
