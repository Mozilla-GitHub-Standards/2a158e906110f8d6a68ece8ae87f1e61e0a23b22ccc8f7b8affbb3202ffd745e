Bookmark Panels

Mozilla Firebird now allows you to open any bookmark as a sidebar panel. This is especially useful for viewing forum indexes, search results, blog post pages, etc. Links are opened in the main browser window.



 

You create a Bookmark Panel by opening the Properties window for any bookmark, and choose "Load this bookmark in the sidebar." When you open the Bookmark from then on, it is opened in a Bookmark panel. All links in the page are opened in the main browser area.
