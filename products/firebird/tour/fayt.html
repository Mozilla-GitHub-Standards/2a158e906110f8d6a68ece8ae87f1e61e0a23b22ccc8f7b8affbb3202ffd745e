Find As You Type




Find As You Type allows you to navigate pages more quickly if you prefer to use the keyboard. Start typing the text of a link and Firebird will select the first matching link. Press Enter to follow the link, or Ctrl+Enter to open the link in a new tab. Press Ctrl+G to advance to the next match. To search in any text in the page (not just links), press "/" before you start to type.
