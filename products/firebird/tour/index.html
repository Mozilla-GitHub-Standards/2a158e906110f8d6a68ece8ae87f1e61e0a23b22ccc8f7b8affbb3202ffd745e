Tab Browsing



Tab Browsing (shown above) lets you load several pages in the background while you read another. While this might seem simple, it allows you to graze through a top level page (such as a new site, as pictured, a forum, or other list of links) and selectively load pages that interest you in the background. These pages can be read once you're done with the main page, reducing back-and-forward navigation. Furthermore, once you're done with each page, the next page is usually done loading, effectively eliminating page load delays. Stop waiting, start surfing!

 



Tab Browsing is also integrated with other parts of the browser, such as Bookmarks. You can save all the currently open tabs into a folder of Bookmarks, and then open them all with a click (see above). You can also set a folder of bookmarks as your Home Page, so when you start the browser the contents of the folder is opened in tabs.
