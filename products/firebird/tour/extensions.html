Browser Extensions

Mozilla Firebird has a comprehensive feature set, but if there are additional tools that you need that aren't included, odds are you'll be able to find them among the numerous free extensions available from Firebird Help.

Extensions enhance the Firebird browser in various ways, each is different. One might provide additional functionality to Tab Browsing, others might offer web developer options. Extensions are installed by clicking an install link and then restarting the browser.



 

Each extension can have its own Options, which are available through the Extensions panel under Tools, Options. If an extension is giving you trouble, you can disable it with a click.
