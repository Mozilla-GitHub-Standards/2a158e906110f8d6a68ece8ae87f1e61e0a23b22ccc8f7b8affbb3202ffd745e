Basic Functional Test Case Description - Frame

Requirements:

- bft_frame3.html
- bft_frame_body.html
- bft_frame_header.html
- bft_frame_index.html
- bft_frame_navpanel1.html
- bft_frame_navpanel2.html

Description:

The purpose of this test is to verify that the page displays all frames correctly. All frames should contain data, and should automatically update when you select a link.

  1. Launch Navigator.
  2. On windows, in the 'File' menu, select 'Open Page.' On Mac, in the 'File' menu, select 'Open' then select 'Location.' Choose the bft_frame_index.html file.
  3. Select the first navigational panel list item. The body frame content should change to reflect the link you selected.
  4. Select the second navigational panel list item. The navigational panel should change to reflect the selected link.
  5. Select the list item from the secondary navigational panel. The content of the body frame should be displayed to reflect your selection.
  6. Select the link from within the body frame. The initial body frame content should be displayed within the entire window.

Link(s):

Frame test page

Expected Results:

Inital frame layout:Two vertical frames.
Left vertical frame:Should span the entire length of the window, and contain a bulleted list containing two links.
Left vertical frame after new panel:The secondary navigational panel should contain a bulleted list beginning with a link followed by plain text.
Right vertical frame:The rignt vertical frame should be divided into two horizontal frames.
Top right horizontal frame:The top horizontal frame is the header frame. It should contain text that states it is the header area.
Bottom right horizontal frame:The bottom horizontal frame is the body frame. It should contain the body content.
Bottom right horizontal frame after nav selection:The content of the body frame should change from displaying the frameset structure to a new page containing a link.
Navigator window after selection:The entire navigator window should display the initial body frame page when you select the link from the secondary body frame page.
