Test Case Description for copy-entities

Purpose

Determine whether HTML entities are properly copied from web pages.

Initial Conditions

None.

Steps/Description

  1. View the below-linked web page.
  2. Copy its content.
  3. Paste it into a text editor, such as WordPad (Windows) or BBEdit (Mac).

Expected Results

After step #3, the character entities should continue to display correctly when pasted out of the browser.
