Test Case Matrix for Imaging

New Feature Tests:

(Please do not report bugs for these testcases, they are not implemented yet

Feature/Function sub-feature
sub-function		 Description Test Case Name
Imaging Resize to window Determine that large Images can be resized to the browser window and will not cause you to scroll in order to view entire image. resize
Imaging Text and image resize Determine that you are not able to resize an image to window when there is also text on the page, this is a stand alone image feature only. text-image
Imaging Resize to original Determine that it is possible to resize an image to it's original size after it has been resized to window size. resize-original
Imaging Resized to window Determine that after resizing an image to it's original size, it is possible to resize to window again. resized-window
Imaging Save Image Determine that you can right click an image and save to your local drive. save-image
Imaging Send in mail Determine that you can right click an image and send it via email. send-mail
Imaging Set image as wallpaper Determine that you are able to set image as wallpaper. wallpaper
Imaging Print image Determine that it is possible to print an image without changing the currently loaded page print-image
