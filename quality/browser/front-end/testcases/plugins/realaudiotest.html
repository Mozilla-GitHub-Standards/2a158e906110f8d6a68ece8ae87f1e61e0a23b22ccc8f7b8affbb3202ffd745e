Test Case Description for testing RealAudio plugin

Purpose

Determine that RealAudio plugin works properly

Initial Conditions

None

Steps/Description

  1. Launch the browser
  2. Click here to play a realaudio file using the EMBED tag. The audio player should open in the browser window and work properly.
  3. Click here to play a realaudio file using the OBJECT tag. The audio player should open in the browser window and work properly.
  4. Real World Pages To Test:
    • http://media2.bmrc.berkeley.edu/video/projects/EdForum/EdForum.html
    • http://abcnews.go.com/sections/us/video_index/video_index.html Click on clips to open Javascript window.
    • http://www.spinner.com/ For PR# 32242. Follow links to launch Spinner Lite.
  5. Click on links below to open and play real audio files:
    1. realaudio file [AUTOSTART=FALSE]
    2. realaudio file [AUTOSTART=TRUE]
    3. realaudio file [AUTOSTART=TRUE]
    4. realaudio file [AUTOSTART=TRUE]
    5. realaudio file [AUTOSTART=TRUE]
    6. realaudio file [AUTOSTART=TRUE]
  6. Go to the IBench website. Click on "START" button on the center of the page, click on "I Agree". When the performance page loads up, select only "audio", "video", select speed and click on "RUN" . Go throught the tests and check the results that appear.
  7. Netscape Radio- Go here. Click on "Launch Player", select channel and verify that sound works.
  8. ESPN Radio Audio Links - Try some audio links on this page
  9. NBA TV - Click here. Try out some video links from "NBA TV" on top
  10. NBA Highlights. - Click here. Try out audio/video links on this page
  11. News.com Videos - Scroll down and try the videos on right side under "News.com Video & Audio". Click on each tab on top of the webpage to check other videos on those pages, Eg : "Enterprise" ,"E-Business" and so on.
  12. C-Span Video Page.
  13. Weather.com - Audio and Video tests
  14. Video test page - Try out video clips on this page (lots of them)
  15. Cnet Radio website
  16. IFilm website.

Expected Results

All Real media files should load and play properly.
