 To select text that has been created in the Ender application. The selection procedure can be accomplish manually or using menu commands(Select All).

Text Selection

  1. Using Text Selection manually
    • Select a word.
    • Select a paragraph.
    • Select all text.
    • Select a discontiguous area of text.
  2. Using Text Selection from the menu command
    • Select All.
  3. Using Text Selection from the keyboard command
    • Select All.
  4. Using Text Selection and Edit Menu commands.
    • Select text and choose Cut from the Edit menu.
    • Select text and choose Copy from the Edit menu.
    • Select text and choose Paste from the Edit menu.
    • Select text and choose Clear from the Edit menu.
