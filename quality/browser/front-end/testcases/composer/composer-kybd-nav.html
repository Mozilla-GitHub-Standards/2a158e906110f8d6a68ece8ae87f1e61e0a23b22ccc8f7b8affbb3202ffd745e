Category: Navigation

Through this procedure, a selection can be dynamically changed by moving the cursor left, right, up or down. Double and Triple clicking will select the entire word or block of text. With the shift key down, the current selection can be extended.

Keyboard Navigation

  1. Left Arrow selection
    • Select a word in a sentence.
      • Press the Left arrow key.
  2. Right Arrow selection
    • Select a word in a sentence.
      • Press the Right arrow key.
  3. Up selection selection
    • Select a sentence in a paragraph
      • Press the Up arrow key.
  4. Down selection selection
    • Select a sentence in a paragraph.
      • Press the Down arrow key.
  5. Extend Selection.
    • Extending a selection up.
      • Select text in a paragraph near the top of the paragraph.
      • With the shift key down, mouse down near the bottom of the paragraph.
    • Extending a selection down.
      • Select text in a paragraph near the bottom of the paragraph. With the shift key down, mouse down near the top of the paragraph.
  6. Double and Triple to create a selection
    • Double click selection
      • Double click on a word.
    • Triple click selection
      • Triple click on a word.
