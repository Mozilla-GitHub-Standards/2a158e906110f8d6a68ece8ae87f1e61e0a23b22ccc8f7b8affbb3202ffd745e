Test Case Description for copy-js-text

Purpose

Determine whether text output from a JavaScript applet is properly selected in conjunction with other page content.

Initial Conditions

None.

Steps/Description

  1. View the attached test case.
  2. Drag-select from "Browser Javascript Test" to the bottom of the document.
  3. From the Edit menu, select "Copy".
  4. View the resulting clipboard. (e.g. "Show Clipboard" from Finder on Mac, or paste into a text editor on Win32/Linux.)

Expected Results

After step #3, the entire content of the page should be copied.

After step #4, the text should be copied reasonably. (use your best discretion, or compare to output of IE 5.)
