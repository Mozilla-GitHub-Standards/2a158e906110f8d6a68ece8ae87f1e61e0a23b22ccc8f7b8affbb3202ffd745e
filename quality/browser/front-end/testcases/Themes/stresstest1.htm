Stress Test Case Description for:  StressTest1.html 

Author: Patty Mac

Purpose: To verify the themes should handle properly while opening the multiple windows and switching themes in the Preferences dialog.


 

Description: 
  1. Open up 5 windows using ‘New Navigator Window’ under File menu.
  2. From the main menu bar, select Edit>Preferences/Appearance/Themes/Classic. 
  3. Click on the "Apply Classic" button.
  4. Close the Preferences dialog.
  5. Check to see if all the multiple windows should apply to Classic theme.
  6. From the main menu bar, select Edit>Preferences/Appearance/Themes/Modern. 
  7. Click on the "Apply Modern" button.
  8. Repeat step 4.
  9. Check to see if all the multiple windows should apply to Modern theme.

Expected Results:

All the opened windows should apply to the new selected theme.
