Test Case Description for Help contents

Purpose:

This basic testcase is designed to verify that Help contents come up when clicked and is displayed corectly

Initial Conditions:

Netscape should be started

Steps/Description:

1. Select the toplevel Help menu.

2. Select Help Contents

3. Select all contents on right, copy and paste into your favorite word processor and run spell check
.

Expected Results:

2. Help contents should come up with main topics on the left and detailed information on the right

3. There should be no spelling errors
