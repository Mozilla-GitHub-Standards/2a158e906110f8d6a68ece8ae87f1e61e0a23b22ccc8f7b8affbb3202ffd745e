Test Case Description for Help Menu Items


Purpose: 
 This basic testcase is designed to verify that Help menu items appear and that when selected, they take you to the appropriate help contents

Initial Conditions: 
 Netscape should be started 

Steps/Description: 
 1. Click the toplevel Help menu once

2. Read the items listed

3. Click on each menu item
.

Expected Results: 
 1. The menu item should stay open

2. Help and Support Center, What's new in Netscape 6.1, Feedback Center, Security Center, About Plugins, and About Netscape should be listed

4. Verify that each menu item selected takes you to the appropriate help content
