Test Case Matrix for Drag & Drop

The matrix lists several test cases for verifying that drag-and-drop operates properly. System integration on Windows, Mac OS, and Linux are also tested.

Active test cases
Feature/Function Description
D&D Links across browser windows Confirm that dragging a link across browser windows opens the link.
D&D Links onto the Desktop Confirm that dragging a link from the browser window onto the desktop creates a shortcut.
D&D Images across browser windows Verify that dragging an image across browser windows opens the image
D&D Images onto the Desktop Vrify that dragging an image onto the desktop saves the image
D&D to Frameset page Verify that dragging onto a frame-set page loads the dragged content properly.
D&D to Composer Verify that browser contents can be dragged to a Composer or Compose Mail window.
D&D to Dialog Confirm that text, links, and images dragged to a non-modal dialog are handled properly
D&D from the desktop – Folders Confirm that dragging a directory to the browser window displays the folder contents
D&D from the desktop – Shortcuts Confirm that dragging an Internet shortcut to the browser window opens the URL
D&D from the desktop – Text files Confirm that dragging a text file to the browser window displays the document contents
D&D from the desktop – Images Confirm that dragging a supported image file to the browser window display the file
D&D from the desktop – Plug-in files Confirm that dragging a helper-application file to the browser window opens the file properly
D&D from the destkop – Invalid Items Confirm that invalid items dragged to the browser window are handled properly
D&D from Find File window Verify that items dragged from a Find File window are handled properly (Windows / Mac OS)
D&D onto the Application Icon Verify that items dragged onto the application icon are handled properly
D&D URL Proxy Icons Verify that dragging the URL proxy icon to a browser window opens that URL in the target window
D&D URL Proxy Icons onto the Desktop Verify that dragging the URL proxy icon onto the desktop creates a shortcut
D&D from Bookmark Manager Determine if the application and OS properly handle items dragged from the Bookmark Manager
D&D to Bookmark Manager Determine if the Bookmark Manager properly handles items dragged onto it.
D&D Large Images (stress) Verify that dragging a large (e.g. 5 MB) image across browser windows, to desktop, and into mail message works properly.
D&D Personal Toolbar Determine if you can drag items to and from the Personal Toolbar
D&D Bookmarks menu to toolbar Verify that you can drag a bookmark from the Bookmarks menu to the Personal Toolbar.
Netscape-only Tests
Feature/Function Description
Dragging links to AIM Verify that you can drag a URL or page content to an AIM window
