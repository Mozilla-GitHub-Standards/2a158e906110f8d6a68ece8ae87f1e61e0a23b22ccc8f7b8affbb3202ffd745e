Test Case Description for dragging into an editor window

Purpose

Verify that links, images and rich text can be dragged from a browser window to a editor window.

Steps/Description

  1. Open the testfile.html test file
  2. Open a Composer window
  3. Drag the Drag Me link from the browser window to the Composer window
  4. Drag the space shuttle image from the browser window to the Composer window
  5. Select the formatted text section, including part or all of the table, and drag the selection from the browser window to the Composer window
  6. From the location bar, drag the URL proxy icon to the Composer window.
  7. Open a HTML Compose Mail window (if the compose window opens in plain text mode, close the window, then in the mail window Ctrl+click (Cmd+click in Mac OS) the Compose button)
  8. Repeat steps 4 to 6 for the Compose Mail window

Expected Results

3-5: The Composer window should accept each content item without any loss of content, formatting, or other attributes.

6. The editor should add a link to the URL displayed in the location bar.

8. The Compose Mail winodw should accept each contents just as Composer would
