COMPONENT

Description

The area I will be testing is the Composer component of Netscape 6.0. I will be primarily testing the user interface of Composer. See "Areas To Be Tested"

Undocumented Features

APPROACH

Strategy

  • Functional testing - this will be performed at the black box leve(see testcases)
  • Regression testing - this will be covered by bug verification/regression testing.
  • Ad-hoc testing - this will be covered by the tester; random feature/function testing.
  • Acceptance testing - this will be covered by daily smoke testing.
  • Unit testing - this will be covered by the engineering team

RESOURCES

  • Windows 98/NT/2000
  • Linux RedHat 6.1

Will use the lab for the OS's I don't have in my cube (i.e. NT, Windows 2000, Mac 8.6 )
