Browser/Composer front-end test plans

by Paul Wyskoczka

Browser and Composer Test Plans? What would I use these for?

The browser/composer front-end QA team publishes these test plans because we don't know everything.

Please feel encouraged to review the test plans presented below, and to contact any test plan authors with cogent feedback on their test plans. We welcome your thoughts on errors and major omissions -- particularly if you're willing to personally revise the test plan to address any deficiencies you identify.

What test plans are currently available?

  • Bookmarks
  • Command-line options
  • Composer
  • Copy & Paste
  • Drag & Drop
  • File downloading and saving UI:
    • Download Manager
    • General downloading UI
  • Form Manager (wallet)
  • Help
  • Helper application UI
  • History
  • Imaging
  • Keyboard navigation
    • Tabbing and focus
    • Scrolling and focus
    • Type ahead find
  • Navigator menus:
    • Toplevel (main) menus
    • Context menus
  • OJI
  • Password Manager (single signon)
  • Plugins
  • Preferences UI
  • Printing
  • Search
  • Sidebar
  • Some cross-platform browser features: Open Web Location, Find in Page, file pickers, View Source, Page Info
  • Text Selection
  • Themes
  • Tabbed Browser
  • Toolbars, etc.(Window Controls, Full Screen Mode)
  • Site Map
  • Security Updates
  • Contact Us
  • Donate
