Browser Buster: an eyeball hour testing project

This is a sanity checking and stress testing system for the browser. The
system loads various lists of pages for as long as the browser continues to
run. Begin the test by clicking the link below.

     top 100 site list - new page loads every 25 seconds.
     random site list - new page loads every 30 seconds. Provides a wider
     variety of sites, content, and error conditions. May not run as long
     and need more tending and restarting.  Thanks to Craig Newell on the
     idea for this test.

Here is the current list of the pages that are tested each cycle and some
other interesting pages that you can load "by hand."

Click here to reset your page counter if you like to count the number of
pages that you have loaded in any one testing session.

Contact chofmann to add test URLs or kick the system if it doesn't seem to
be responding.

Other Tests

To test page size varitation and its effect on memory consumption and speed
you can use the dial-a-page-length cgi below. Insert the number of lines of
content you want to test in the browser in the box below:

  lines long.

Or click on these links for pages of about .5, 1, 2, 4, 8, and 16 Megabytes.
