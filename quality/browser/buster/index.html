Browser Buster: Chris Hofmann's eyeball hour testing project

This is a sanity checking and stress testing system for the browser. The system loads various lists of pages for as long as the browser continues to run. Begin the test by clicking on one of the links below.

    top 100 site list - new page loads every 25 seconds.
    International top 100 site list - new page loads every 27 seconds.
    random site list - new page loads every 30 seconds. Provides a wider variety of sites, content, and error conditions. May not run as long and need more tending and restarting.  Thanks to Craig Newell on the idea for this test.
Here is the current list of the pages that are tested each cycle and some other interesting pages that you can load "by hand."

Click here to reset your page counter if you like to count the number of pages that you have loaded in any one testing session.

If you spot a problem with a site or the ability to run the test, post a description of the problem in the browser buster tracking bug (bugzilla 197425)and chofmann or one of the browser buster maintainers will change a URL or kick the system if it doesn't seem to be responding.
