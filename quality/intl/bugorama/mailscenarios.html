International Bug-o-Rama: Mail Scenarios


Written by: Katsuhiko Momoi
URL of this document: http://www.mozilla.org/quality/intl/bugorama/mailscenarios.html
URL of the Browser  scenarios: http://www.mozilla.org/quality/intl/bugorama/browserscenarios.html

After trying out one or more of the following scenarios, please report any bugs or problems you find to:

Mail: Katsuhiko Momoi <momoi@netscape.com>. Also all Japanese messages to momoi.
Brower & Composer: Allan Masri <amasri@netscape.com>. Non-Japanese only.

Mail consultant: Kat Momoi (see you at irc.mozilla.org, Channel name: #i18nbugs)

Preliminary Steps:

  • Use a Mac, Win, or Unix build from 3/6/2000 or later. (Do not use a build prior to 3/6/2000). Start here:
    • ftp://ftp.mozilla.org/pub/mozilla/nightly/
  • If you see more than one builds for that day, take the last one of the day for your platform.
  • Get the installer file. Then run it.
  • When the installation is done, on some platforms, Mozilla will start automatically. On others, you need start it "mozilla.exe".
  • If you're a first time user, you will be asked to create a new profile. Do this and when it starts up, choose "Task | Mail" to bring up Messenger.
  • You will then be asked to create a new Mail account to use. You can also manually get to Account Setup via the menu, "Edit | Mail/News Account Settings". You can set user ID, password, server name, type of server (IMAP, POP, News, Local). You can experiment with various other settings in the Settings window and see if they work.
  • We don't provide detailed menu items -- please explore these yourself. You should find out on your own what is available and what is not.


Pick one or two scenarios of your choice from below. Or come up with your own scenario to try out.

Test Scenario 1: Replying to a message written in a language you don't understand

  • You have received a letter from a foreign customer (French, German, or Portuguese). Pick a language that you don't understand. Pick up the text of the letter at:
    • http://www.mozilla.org/quality/intl/bugorama/data/
  • You want to respond to this but you don't know what this letter means. Go to a translation site, e.g.:
    • http://world.altavista.com/
  • Get rough translation of this letter, compose a reply and translate it into the target language, paste in the original,
    • and send it to me at: iqax2@netscape.com.
    • Note: It's up to you to decide whether or not you should send HTML or Plain text letter.


Test Scenario 2: Sending a page to a friend

  • You are surfing the net and come upon an interesting page, which is not in English.
  • You remember that a friend of yours can read this language and might be interested in it. So you send your friend this page.
  • Try selecting at least 2 different pages. Try, for example, one of the Netscape International Home Pages. But also try sending some other pages from the net.
    • http://home.netscape.com/ja
    • http://home.netscape.com/ko
    • http://home.netscape.com/zh/cn
    • http://home.netscape.com/zh/tw
    • http://home.netscape.com/de
    • etc.
  • When you send a page, write some comments about the page in English in one case and in another language.
Test Scenario 3: Sending messages in different languages: copy/paste
  • Find yourself different language web pages
  • Using copy/paste method, send yourself messages in different languages. (You need to use View | Character Set menu on the Mail Composer).
  • Use your Address Book to pick out friends to send out your messages to.
  • Are messages displaying correctly in these languages?
  • Are there any difficulties in doing this?
Test Scenario 4: (News reading and posting)
  • Subscribe to some newsgroups you normally read on your internal news server. Try reading articles in non-English newsgroups.
  • After reading several messages, look at the "Task | Address Book" and in particular "Collected Addresses" and see if the addresses of the senders have been added there.
  • There is a public news server outside Netscape. news.mozilla.org. Post some international messages using test news groups: netscape.public.test
Test Scenario 5: Using various options at Send time.
  • Compose a message  in any language you like.
  • But before sending this message, look under "Options" menu and use some of the options listed there.
