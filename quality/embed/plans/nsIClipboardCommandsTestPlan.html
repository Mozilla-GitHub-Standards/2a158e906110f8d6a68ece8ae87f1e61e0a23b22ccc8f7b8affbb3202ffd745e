Test Plan for nsIClipboardCommands interface.

    nsIClipboardCommands interface provides methods to do cut/copy/paste operations on the
    current selection within the webbrowser window.
    In order to test these methods you need to first get an instance of nsWebbrowser.

    nsCOMPtr<nsIWebBrowser> mWebBrowser;
    mWebBrowser = do_CreateInstance(NS_WEBBROWSER_CONTRACTID, &rv);

    Once you get an instance of nsWebbrowser you can get nsIClipboardCommands object
    by doing a GetInterface on the webbrowser object.

    nsCOMPtr<nsIClipboardCommands> clipCmds = do_GetInterface(mWebBrowser);

    Now you can call the methods using the clipCmds object.

    To see the test cases click on the desired method below.

    canCutSelection()
    canCopySelection()
    canPaste()

    copySelection()

    selectNone()

canCutSelection()

    Introduction:
    Returns whether there is a selection and it is not read-only.

    How to use:
    PRBool canCutSelection = PR_FALSE;
    nsCOMPtr<nsIClipboardCommands> clipCmds = do_GetInterface(mWebBrowser);
    if (clipCmds)
        clipCmds->CanCutSelection(&canCutSelection);
    if (canCutSelection)
        AfxMessageBox("The selection you made Can be cut");
    else
        AfxMessageBox("Either you didnot make a selection or The selection you made Cannot be cut");
 
Case  Steps Expected Results Pass/Fail Comments
1 Make a selection of some text on the page and click cancutselection. Should return "The selection you made cannot be cut" since you cannot cut text on web page. pass
2 Type something in the yahoo search text box and select the contents in that  text box and click cancutselection. It should return "The selection you made can be cut" since you can cut this text. pass
3 Write a small html pgm containing a form which contains a text area with read only text.Select    some content from the text area and click on cancutselection. It should return "The selection you made cannot be cut" since the text is read only and you cannot cut it. pass

canPaste()

    Introduction:
    Returns whether the current contents of the clipboard can be pasted and if the current selection
    is not read-only.

    How to use:
    PRBool canPaste = PR_FALSE;
    nsCOMPtr<nsIClipboardCommands> clipCmds = do_GetInterface(mWebBrowser);
    if (clipCmds)
        clipCmds->CanPaste(&canPaste);
     if (canPaste)
         AfxMessageBox("The clipboard contents can be pasted here");
     else
         AfxMessageBox("The clipboard contents cannot be pasted here");
 
Case Steps Expected Results Pass/Fail Comments
1 Write a small html pgm containing a form which contains a text area with read only text. make a   selection on the text area and click 
canpaste.(make sure that clipboard contains some data).		 Should return "The clipboard contents cannot be pasted here" since you cannot paste into a read-only text area. pass
2 Go to yahoo search text box and place the cursor in text box.then click on canpaste. should return "The clipboard contents can be pasted here" since this is not read-only text box. pass

cutSelection()

    Introduction:
    Cut the current selection onto the clipboard.

    How to use:
    nsCOMPtr<nsIClipboardCommands> clipCmds = do_GetInterface(mWebBrowser);
    if (clipCmds)
        clipCmds->CutSelection();
 
Case  Steps  Expected Results Pass/Fail Comments
1 Type something into a text box and cut it and try to paste it into another text box. If it pastes it into another text box that means it was cut successfully.The first text box should not contain any  text since it is cut. pass
2 Type something into a text box and cut it and try to paste it into a notepad. It should paste into notepad.The first text box should not contain any  text since it is cut. pass

selectNone()

    Introduction:
    Clear the current selection (if any). Insertion point ends up at the beginning of current selection.

    How to use:
    nsCOMPtr<nsIClipboardCommands> clipCmds = do_GetInterface(mWebBrowser);
    if (clipCmds)
        clipCmds->SelectNone();
 
 
Case Steps Expected Results Pass/Fail Comments
1 Click on selectnone. It should deselect whatever selection you made. pass
2 Click on selectnone. In addition to deselecting whatever selection you made it should put the insertion point at the location from where selection was made. pass
