Future projects:

Networking - full component review (after Mozilla 1.0)

Currently underway. A bug-by-bug review has reached about bug #3000.
A general low-hanging fruit query is focused on RESOLVED/DUPLICATE.
Commentary:
Networking is clearly a parking lot for other components. Many bugs that are dupes are actually depends, and need to be tested by the QA for the affected component. HTTP's growth is rapid, possibly due to the introduction and debugging of HTTP Pipelining.

Updates:
Mozilla 1.2 (final) Mozilla 1.3a is coming soon, so the queries were run to reflect pre-bug review totals.
