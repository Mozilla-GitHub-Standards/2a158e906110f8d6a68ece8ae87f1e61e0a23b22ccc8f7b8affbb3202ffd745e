Networking (general) Test Plan

Abstract:
This document provides general network functional testing. The test area covers essential networking functions, and can be considered a "smoketest" of networking functionality.Full functional testing should be done on a per-protocol basis.

This document is written for testing mozilla-based products, but has been structured so other browsers can be easily tested.
Results can be summarized in the results template.

Overview:
The majority of the tests are link-clicks. The results include numerous dialog boxes, so automation is not possible.

Previously, tests and results were integrated. As of Mozilla 1.3b, test results can be recorded in a template, to provide better summarization.
