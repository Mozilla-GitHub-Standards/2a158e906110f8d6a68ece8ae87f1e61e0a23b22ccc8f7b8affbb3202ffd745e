The testcases are currently written for the Mozilla trunk. Currently, three main testcases exist:

coretests
filetests.html
ftptests.html

The number of minor tests, like data: URLs is finally reaching completion.

The current testcases can be found in http://www.mozilla.org/quality/networking/testing/

Like the trunk, the testcases are constantly being improved and modified. When a release is tested, the current tests are updated, and then copied to the results folder for the release (Mozilla 1.0, Chimera 0.5, etc). When the tests are run against a specific build, the results are saved to the template file and the template is copied into a file named after the build date.

Release specific comments are maintained in red, to hilight the differences.

The tests currently integrate results for the primary platforms (Win32, Linux, Mac OS X, Mac OS classic).
