 Configurations: The following target configuration were tested.

HP Vector VL Pentium II 266mhz with 128 mb of ram running Windows 98 SE

HP Vector VL Pentium II 266mhz with 128 mb of ram running Red Hat Linux 6.2

Power Mac G3 266 mhz with 128 mb of ram running MacOS 9.0


Setup:
  1. Install the ns mojo using the recommended installation
  2. Remove all profiles and start the browser with the only one profile so the profile manager is not  encountered
  3. Turn the sidebar off in the view menu
  4. In the preference panel set the first page to be displayed to about blank.
  5. Reboot the machine before the launch timing is taken

 Notes:
  1. * There was no build for Linux on 2/28, so I used 2/15 which was the closed build I could find.
  2. ** This timing seems a little suspect to me.  Have to try this on other machines and reinstall on target Mac
  3. For each timing run 5 times and throw out the high and low values.  Average the remaining values for final result.
