Mail and News Component: Address Book Import/Export

Written by Esther Goes

References:

UI Specifications: Address Book Specification (Communicator 5.0 Integrated Mail and Address Book Import utilities

Address Book Import: Available in file menu of the Address Book (or Tasks|Tools|Import)

  1. Visual Dialog: Title -> Import
  2. Sub-title: Netscape 6 Mail Import Wizard; Import Mail and Address Books from other programs
    1. text description "This wizard will import mail messages, address books entries and/or preferences from other mail programs and common addressbook formats into Netscape 6 Mail."
    2. list box titled "Select the type of material to import
      1. Address Book
      2. Mail
      3. Settings
    3. actions buttons: Back,Next & Cancel; verify all these buttons work; verify the close window widget works
    4. Click on Next to display second dialog
  3. Second Visual dialog, after selecting the Address Book radio button
    1. description box "Please select the program from which you would like to import:
      1. Communicator 4.x
      2. Eudora
      3. Outlook
      4. Outlook Express
      5. Text
    2. actions buttons: Back,Next & Cancel;
      1. verify clicking Cancel the process is stopped and dialog closes
      2. verify clicking Next brings up the correct dialogs for you to select the
      3. verify clicking Back returns to the previous dialog
      4. verify the close window widget works
  4. Communicator 4.x: Selecting Communicator 4.x imports the Comm 4.x address book
    1. verify a new address book appears with the name of the Comm 4.x address book
    2. verify all the cards are present
    3. verify the contents of each card is correct
    4. verify all the mailing lists are present
    5. verify the contents of each mailing list is correct
  5. Eudora: Selecting Eudora imports a Eudora address book
