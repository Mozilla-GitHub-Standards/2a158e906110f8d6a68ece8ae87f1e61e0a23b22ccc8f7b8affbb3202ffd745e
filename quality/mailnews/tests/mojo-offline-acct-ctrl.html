Feature Test Spec: Mail and News Component

Mail and News Component: Offline: Account Central

Written by Gary Chan

References:

UI and/or Functional Specifications: http://www.mozilla.org/mailnews/specs/offline/

The test cases outlined in this document are intended to cover overall mail and news with respect to the Account Central page that is present when you select a mail or newsgroup server. This covers only testing as it relates to Offline and is not intended to cover all features within Account Central.

  1. General UI tests: covered under separate test spec Offline: General UI.
  2. IMAP/Webmail/POP test
    1. Clicking on the following links, while offline, it behaves as expected with no adverse effects.
      1. Read messages
      2. Compose a new message
      3. View settings for this account
      4. Create a new account
      5. Search messages
      6. Create message filters
      7. Offline Settings (only present for imap/webmail)
  3. Newsgroup test
    1. Clicking on the following links, while offline, it behaves as expected with no adverse effects.
      1. Subscribe to newsgroups
      2. View settings for this account
      3. Create a new account
      4. Search messages
      5. Offline Settings
  4. Local Folder test
    1. Clicking on the following links, while offline, it behaves as expected with no adverse effects.
      1. View settings for this account
      2. Create a new account
      3. Search messages
