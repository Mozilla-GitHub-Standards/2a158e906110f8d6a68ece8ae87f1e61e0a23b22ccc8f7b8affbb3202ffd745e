Feature Test Spec: Mail and News Component

Mail and News Component: Address Book/Print

Written by Ninoschka Baca

References:

  • UI Specification: http://mozilla.org/mailnews/specs/addressbook/AB_Menus.html
  • Other: How To Create Address Books, Cards, Lists, Directories, etc.

Contents

  1. Print Cards
    1. Print one card
      1. Verify with an address book selected in the directory pane and a card selected in the results pane, that one card prints
      2. Verify with a list selected in the directory pane and a card selected in the results pane, that one card prints
  1. Print multiple cards
    1. Address Book selected in the directory pane
      1. Verify using the Ctrl key to multi-select cards prints continuously (i.e. prints 2 cards continuously, not one page per card)
      2. Verify using the Shift key to multi-select cards prints (i.e. print 3 cards continuously, not one page per card)
    2. List selected in the directory pane
      1. Verify using the Ctrl key to multi-select cards prints continuously (i.e. prints 2 cards continuously, not one page per card)
      2. Verify using the Shift key to multi-select cards prints (i.e. print 3 cards continuously, not one page per card)
  1. Print Lists
    1. Print one list
      1. Verify with an address book selected in the directory pae and a list selected in the results pane, that a list prints
      2. Verify with a list selected in the directory pane, that a list prints
    2. Print multiple lists
      1. Verify with an address book selected in the directory pane and multiple lists selected (Ctrl or Shift) in the results pane, that the lists print continuously
      2. Verify with multilple lists selected (Ctrl or Shift) in the directory pane, that the lists print continuously
  1. Print Address Books
    1. Verify with an address book selected in the directory pane that the address book prints correctly (cards/lists print continuously)
    2. Verify with multiple address books selected in the directory pane that the address books print correctly
