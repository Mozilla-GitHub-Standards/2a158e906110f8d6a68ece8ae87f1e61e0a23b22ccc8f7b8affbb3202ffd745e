Context Menus Check List for 3-Pane Window
 

Created: 02/15/2002 by olgam


Related links:  

Summary: Context menus (Pop-up menus) provide an efficient way for the user to access the operations of objects. Pop-up menus populated with commands (actions) specific to the object or its immediate context - it reduces the number of commands to browse through.
