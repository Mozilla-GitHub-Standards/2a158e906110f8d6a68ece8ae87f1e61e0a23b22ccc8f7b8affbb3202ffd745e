Feature Test Spec: Mail and News Component

Mail and News Component: Layout - CSS

Written by Esther Goes

Description

This is to test that CSS1 compliance documents, when attached or send page via mail, are displayed correctly when viewing in Mail and News messages.

Compatibility

  1. Compatibility with messages sent from Netscape 6.2.2 (randomly select 10 of the test cases to Send to MachV)
    1. Windows release
    2. Mac release
    3. Unix release
  2. Compatibility with messages sent from previous versions of Communicator
    1. Communicator 4.79
  3. Compatibility with messages sent from Microsoft Applications
    1. Outlook Express
    2. Outlook 98
  4. Compatibility with messages sent from Eudora (current version, 4.1)

CSS Matrix testing

(Use test cases from the CSS1 Matix located on Mozilla.org underTesting|browser standards compliance qa. Warning: there are many tests to run (400+). Here are the different approaches being considered.

  1. For each test case found in the CSS1 Matrix table, copy the URL for each test case and attach the Web Page in new HTML message compose window as an attachment. Address message to yourself.
    1. Copy the URL into the Web Page attachment dialog and send. Verify that the page displays the same in the stand alone Message Pane
    2. Copy the URL into the Web Page attachment dialog and send. Verify that the page displays the same in the Message pane of the 3-Pane Window
  2. For each test case found in the CSS1 Matrix table, Send Page to yourself. Verify that the page displays the same in the Message pane of the 3-Pane Window as it does in the Brower page.
    1. Open the URL in the Browser, File|Send Page to youself. Verify that the page displays the same in the stand alone Message Pane
    2. Open the URL in the Browser, File|Send Page to youself. Verify that the page displays the same in the Message pane of the 3-Pane Window
