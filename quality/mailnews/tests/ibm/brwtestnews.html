Newsgroup Accounts: Add Newsgroup Account / Server
 

Action
Expected result
Open Newsgroup window by clicking Mail menuitem under Tasks menu in the mozilla browser. A separate Mail / Newsgroup window has to be opened.
Click on the menu item Mail/News Account settings  under Edit menu Account settings dialog box has to be opened.
Click on New Account  tab. Account wizard should be displayed.
Select a Newsgroup account  radio button and click on next. New dialog box should appear for accepting name and email address.
Enter your name as 'India' and email address as 'test10@netsdom.com', then press next New dialog box should appear for accepting news server name.
Enter the sever name as 'ausnews.austin.ibm.com', then press next. A dialog box should appear for accepting an account name.
Enter account name as 'Account1', then press next. A new dialog box should appear saying 'congratulations!, you have enterd all the required information...........'.
Click on Finish tab. A folder should be included in the mail folders list with the name 'test10 on ausnews.....'
Click on OK tab in the Account settings window. Verify that the newsgroup account 'Account1' is added in the Mail folders list.

Newsgroups: Search Messages: Single match
 

Action
Expected Result
Open Newsgroup window by clicking Mail menuitem under Tasks menu in the mozilla browser. A separate Mail / Newsgroup window has to be opened.
Select a newsgroup 'netscape.public.mozilla.os2' or some other newsgroup under account 'Account1' in the Mail folder list and press right mouse button.Then select Search Messages... option. Search Messages dialog box should appear. And 'netscape.public.mozilla.os2' or any other selected newsgroup should appear in Search for items in text box.
      Select Match any of the following or Match all of the following radio button(both result same for single match).
      Select sender or subject or any item in the list box for searching string to be present.
      Select any of the item( contains, is, begins with, ends with) in the second list box, the searching string to be matched(i.e  start with string or end with string or string in middle or exactly as string). 
      Enter the string to be searced in edit box. 
      Press Clear search tab(incase of  subsequent search)
      Press Options tab.		 Search options dialog box should be displayed.
Enable the check box to search even in sub-folders. Select radio button search the specified server.
Then press OK tab in the Search options dialog box .		 Search options dialog box should disappear and should return to Search Messages dialog box.
Press Search tab. Verify the resulted messages in the bottom of dialog box are according to the search format or not.
(No messages result for unmatched search)
Select the resulted messages and press File messages.... tab.  A list of folders should display.
Then select the folder in the list displayed to place the selected messages.  Verify that the messages placed in the selected folder.
(Those messages are no longer exist in the search dialog)

Expected Result

Notes:-

Signature file: It is a text file containing address information or something like that.
Thread: It contains a list of messages. Those messages are original message and replies to original message or reply to reply message and so on.....

Unread Thread: Any thread contains atleast one unread message is called unread thread.

Watched Thread: A thread which has a symbol  like binocular in the thread column at message list.
Headers: These are sub-titles like Subject, From, Organisation, Date, Newsgroups, Path, Lines, Message-ID, NNTP-Posting-Host, Mime-Version, Content-Type, Content-Transfer-Encoding, X-Mailer, X-Accept-Language, Xref in the message apart from the actual matter.
