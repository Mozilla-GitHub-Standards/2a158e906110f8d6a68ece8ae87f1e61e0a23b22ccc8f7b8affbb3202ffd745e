Appearance:Appearance
 
Action Expected Result
From the Edit Menu of the Browser , select preferences The preferences window should get displayed.
Select the category Appearance in the preferences window The Appearance category should get highlighted.
Select item Navigator under On startup, launch and restart the  mozilla browser. The Mozilla should start with Browser window.
Repeat the above steps by selecting Messenger , Composer and Calender. Even a combinations of these can be selected.
Restart the Mozilla browser.		 Verify that the selected window/ windows gets displayed when you re-start Mozilla browser
In the Appearance page of the preferences window, select the radio button Pictures and text under show tool bar as . Close the preferences window by clicking on OK The browser should display both pictures and text in the toolbar.
In the Appearance page of the preferences window, select the radio button Pictures only under show tool bar as.Close the preferences window by clicking on OK The browser should display only Pictures in the toolbar.
In the Appearance page of the preferences window, select the radio button Text only under show tool bar as.Close the preferences window by clicking on OK The browser should display only Text in the toolbar

 
 

Appearance:Font
 
Action Expected Result
From the Edit Menu of the Browser , select preferences The preferences window should get displayed
Select the item Fonts under the  category  Appearance in the preferences window The Fonts items  should get highlighted.
Select Font type,Serif, San-serif,Cursive,Fantacy,Monospace required under the Fonts for ,Serif, San-serif , Cursive,Fantacy,Monospace dropdowns respectively .
Select the variable width and Fixed width font sizes.
Under the options When a webpage chooses its own font select Use my default fonts, ignoring the fonts page has chosen. Load a web page .		 The fonts type of the web page should be displayed to the one you set in the preferences window and also the sizes of fixed width font and variable width font should be the one you set.
Under the options When a webpage chooses its own font select Use all the fonts chosen by the page ,including dynamic fonts. Load a web page The web page should load its own Font and sizes.

Navigator:Navigator
Action Expected Result
From the Edit Menu of the Browser , select preferences.
Select the category Navigator in the preferences window		 The Navigator category should be highlighted
Select the radio button Blank page under navigator starts with.Close the preferences window and restart the browser The Browser should display a blank page at startup.
Select the radio button Home page under navigator starts with.Enter the home page as "www.mozilla.org" in the location field under Home page Close the preferences window and restart the browser The Browser should display the page "www.mozilla.org" at startup.
Visit a web site say www.ibm.com.
Select the radio button Home page under navigator starts with. Under home page click on Use current page which will make the current page ie www.ibm.com  as the home page.Close the preferences window and restart the browser		 The Browser should display the page "www.ibm.com" at starup.
Select the radio button Home page under navigator starts with. Under home page Select Choose File and select a local file to act as home page.Close the preferences window and restart the browser The Browser should display the page local file selected as the home page.
Click on a link in the page "www.mozilla.org".
Select the radio button Last page visited under navigator starts with in the preferences window.Close the preferences window and restart the browser		 The last visited page should be displayed.
In the preferences window , under the  History section of Navigator category , give 2 days in the field Pages in History expire after. After verifying that some entries are left out in the history folder, close the browser.
Change the system date to 3 days ahead.
Open the Mozilla browser again.		 The history folder should be empty.Verify this by clicking on History under the submenu Tools of the menu item Tasks
In the preferences window , under the Navigator category click on clear history.. The history folder should be empty.Verify this by clicking on History under the submenu Tools of the menu item Tasks.
In the preferences window , under the Navigator category click on clear location bar.. The entries in the location bar should be empty.

Mail and News:Mail and News
 
Before going ahead testing the Mail and News , one have to set an Account for Mail and News.
To set an account for Mail :
  • Open Account Setup.. window by clicking on the Menu item Account setup under Edit menu
  • Click on New Account which will open a New Messenger Account Wizard Window (This option is available in Mozilla mail only - not in Mozilla Navigator)
  •  Select Mail radio button and then click next.
  • In the next page enter your name and a valid e-mail ID.Click next
  • Select pop mail server as the server type and enter the server name as bv3b4.austin.ibm.com for both incoming and outgoing mail Server name field.Click next.
  • Enter India1 as the user name ( whose password is india1) and then click next.
  • Give the account name as Account1  and then click  next.
  • Click on Next after verifying the username and e-mail information displayed.
To set an account for News:
  • Open Account Setup.. window by clicking on the Menu item Account setup under Edit menu
  • .Click on New Account which will open a New Messenger Account Wizard Window (This option is available in Mozilla mail only - not in Mozilla Navigator)
  •  Select News radio button and then click next.
  • In the next page enter your name and a valid e-mail ID.Click next
  • Select News server as the server type and enter the server name as news.mozill.org  for  incoming and bv3b4.austin.ibm.com as outgoing (smtp) in the  Server name field.Click next.
  • Enter India2 as the user name ( whose password is india2) and then click next.
  • Give the account name as Account2  and then click  next.
  • Click on Next after verifying the username and e-mail information displayed.
To add a news group:
  • In the Mail Folders panel , highlight the News server news.mozilla.org.
  • Click on File->Subsrcibe to..
  • In the subscribe to newgroup field enter netscape.public.mozilla.os2 and enter OK.
  • Verify that this news group is added under the  news server news.mozilla.org in the Mail folders panel.

 
 
Action Expected Result
Click on Mail and Newsgroups  in the preferences window The item Mail and Newsgroup should get highlighted.
Select  'Fixed width font' Radio button. Close the preference window and open a mail in the inbox which contains plain text messages. The plain text should be display in 'Fixed width font'.
Select  'Variable width font' Radio button. Close the preference window and open a mail in the inbox which contains plain text messages The plain text should be displayed in 'Variable width font'
