Feature Test Spec: Mail and News Component

Mail and News Component: Import Mail/Settings

Written by Ninoschka Baca

References

Mozilla UI Specifications: http://mozilla.org/mailnews/specs/
Mozilla 3-pane Specification: http://mozilla.org/mailnews/specs/threepane/

This test plan is for Importing Mail/Settings from third party applications such as Eudora, Outlook, Outlook Express and Comm 4.x Local Folders. No specific specification exists for Importing so this test plan covers obvious functionality. As more information is learned then it will be updated. This test plan does not cover importing of the Address Book, which is covered in a different specification.

Windows: Supports importing from Eudora, Outlook, Outlook Express and Comm 4.x Local Folders
Mac: Supports importing from Eudora
Linux: Supports Comm 4.x Local Folders

  1. Visual
    1. Title bar
    2. Content
    3. Radio buttons
    4. Buttons
  2. Functionality
    1. Entry Points
      1. Verify you can open the Import Wizard from Mail
        1. Select File|Import
        2. Select Tasks|Tools|Import Utility
      2. Verify you can open the Import Wizard from the Browser
        1. Select Tasks|Tools|Import Utility
    2. Functionality of Import Wizard Dialogs
      1. Select various elements with the mouse (radio buttons, buttons, options)
        1. Progress forward through the dialogs
        2. Go back through the dialogs
      2. Select various elements using the keyboard (radio buttons, buttons, options)
      1. Progress forward through the dialogs
      2. Go back through the dialogs
    3. Specific Scenarios
      1. Eudora
        1. Import Mail
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Mail radio button, Next
          3. Select Eudora, Next
          4. Check that a Eudora entry appears and all its subfolders appear under Local Folders
          5. Select Edit|Mail/News Account Settings, and there should be a Local Folders entry and no extra entry specific to Eudora.
        2. Import Settings
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Settings radio button, Next
          3. Select Eudora, Next
          4. Select the appropriate configuration file if prompted (Windows: EUDORA.INI, Mac: Eudora Settings)
          5. Check that a Eudora entry appears in the folder pane
          6. Select Edit|Mail/News Account Settings, select the account that was imported from Eudora and it should display atleast 3 panels for the account (Account Settings, Server Settings, Copies and Folders Settings)
            1. Verify that the settings in Eudora are saved here (i.e. Leave messages on server)
          7. Go back to the Main Mail window and retrieve messages using the account imported from Eudora
          8. Send messages from the Eudora account
      2. Outlook
        1. Import Mail
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Mail radio button, Next
          3. Select Outlook, Next
          4. Check that an Outlook entry appears and all its subfolders appear under Local Folders
          5. Select Edit|Mail/News Account Settings, and there should be a Local Folders entry and no extra entry specific to Outlook.
        2. Import Settings
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Settings radio button, Next
          3. Select Outlook, Next
          4. Check that a Outlook entry appears in the folder pane
          5. Select Edit|Mail/News Account Settings, select the account that was imported from Outlook and it should display atleast 3 panels for the account (Account Settings, Server Settings, Copies and Folders Settings)
            1. Verify that the settings in Outlook are saved here (i.e. Leave messages on server)
          6. Go back to the Main Mail window and retrieve messages using the account imported from Outlook
          7. Send messages using the account imported from Outlook
      3. Outlook Express
        1. Import Mail
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Mail radio button, Next
          3. Select Outlook Express, Next
          4. Check that an Outlook Express entry appears and all its subfolders appear under Local Folders
          5. Select Edit|Mail/News Account Settings, and there should be a Local Folders entry and no extra entry specific to Outlook Express.
        2. Import Settings
          1. Start the Import Wizard (File|Import or Tasks|Tools|Import Utility)
          2. Select the Settings radio button, Next
          3. Select Outlook Express, Next
          4. Check that a Outlook Express entry appears in the folder pane
