Editable Sticky Notes

The following demo uses XBL to create a widget that functions as a board for PostIt notes. We create a widget for the board. Its child divs are the notes. In this example, we create a -moz-binding that enables the user to define a board and to supply notes that can be dragged around on the board.

This example extends the original XBL demo to use document.designMode to allow the stickies to be edited directly.
