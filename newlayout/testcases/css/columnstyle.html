Style Applied to Table Columns

Below is a table with 5 columns defined using 5 COL elements inside a COLGROUP. Each COL looks like this:

<col width=100 style="background-color:(varies for each col);">

This means that each column should be 100 pixels wide each, with different styles.

Style Applied to <col> with span Attribute Set

Below is a table with 5 columns defined using 2 COL elements inside a COLGROUP. The COL elements look like this:

<col width=100 span=2 style="background-color: red;">
<col width=100 span=3 style="background-color: blue;">

This means that each column should be 100 pixels wide each. The first 2 columns should have a red background, while the next 3 columns should have a blue background.

Style Applied to <colgroup> with span Attribute Set

Below is a table with 5 columns defined using 2 COLGROUP elements. The COLGROUP elements look like this:

<colgroup span=3 width=75 style="background-color: red;">
<colgroup span=2 width=150 style="background-color: blue;">

This means that the first 3 columns should be 75 pixels wide each, with red backgrounds. The next 2 columns should be 150 pixels wide each, and have a blue background.

Green Blue Yellow Purple Orange
