Printing & Print Preview Check List


  • Printing
    • Print a page
    • Print Frameset "AsIs"
    • Print Framset "each separately"
    • Click on frame and print selected frame
    • Selection
      • Click on an image print "Selection" (should print just the image)
      • Click on IFrame and print the Selection (should print IFrame as it's own document)
      • Select some text in Document, Print selection
      • Select text in a FrameSet Frame, Print Selection
      • Select text in an IFrame in a FrameSet Frame, Print Selection
    • MailNews
      • Print a Mail message
      • Print multiple mail messages
      • Print Address Book Card
      • Print Address Book
    • Editor
      • Open Editor, Print
  • Open Page Setup
    • Change various settings, print page again
    • Test margins, etc.
  • Print Preview
    • Go into PP
    • Test #1
      • Print page
    • Test #2 - Tests Page Setup from PP
      • Open Page Setup
        • Change Orientation, press OK (PP should change)
      • Print
      • Open Page Setup
        • Change Orientation, Cancel (PP Orientation should not change)
      • Print (Orientation should not change)
    • Test 3 - Tests Shrink To Fit (STF) and Scaling from PP
      • If you start with it on STF
      • Select 50% (should scale)
      • Select Custom, enter 75%, press OK (should scale to 75%)
      • Select 100%
      • Select Custom, change the value to 50%, press Cancel (nothing should happen and the combobox should show 100%)
      • Select STF
      • Select Custom, change the value to 50%, press Cancel (nothing should happen and the combobox should show STF)
      • Open Page Setup, STF should be checked, close it
      • Select 100%
      • Open Page Setup, scale should be 100% and STF unchecked
      • Select Custom and enter 91%
      • Open Page Setup, scale should be 91% and STF unchecked
      • When Scaling changes make sure the number of Pages in the PP toolbar changes
    • Test 4 - Orientation
      • Change Orientation and then Open Page Setup to see if it is updated
      • Change it in the Page Setup and see if the toolbar gets updated.
    • Test 5 - Test PP Page Navigation
