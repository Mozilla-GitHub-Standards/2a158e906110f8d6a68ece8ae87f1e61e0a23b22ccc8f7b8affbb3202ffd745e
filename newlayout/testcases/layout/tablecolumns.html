colgroup with span attribute

Below is a table with 2 rows, each row with 5 cells (first row has TH's). There is also a colgroup element:

<colgroup span=5 width=100>

This means that the table has 5 columns, and each column should be 100 pixels wide. Because the columns are defined "up front" with width data, the table should be displayed incrementally, as the rows are downloaded over a modem. This improves perceived performance.

multiple cols with span attribute

Below is a table with 5 columns defined using 2 COL elements inside a COLGROUP. The COL looks like this:

<col width=75 span=3>
<col width=150 span=2>

This means that the first 3 columns should be 75 pixels wide each, and the subsequent 2 columns should be 150 pixels wide each. Because the columns are defined "up front" with width data, the table should be displayed incrementally, as the rows are downloaded over a modem. This improves perceived performance.
