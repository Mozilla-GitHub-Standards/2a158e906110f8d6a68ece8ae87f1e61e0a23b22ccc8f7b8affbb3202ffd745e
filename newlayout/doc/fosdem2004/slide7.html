The procedure

  1. Download the Web page that shows the bug to your local machine. Use the "Web Page, Complete" option in the save dialog to download all linked files, including JavaScript (.js), CSS (.css), frameset (.html), and image files.
  2. Search for <layer>, document.layers, document.all in the source and verify that it is not a techevangelism.
  3. Look for the doctype of the page. Switch between standard and quirks-mode, does the bug persist?
  4. Testcases should be in standards mode.
  5. Remove as much as possible code.
