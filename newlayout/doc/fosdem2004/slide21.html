The Layout Breakpoint

  • Cause incremental reflows by placing
    <script>var v = document.body.offsetHeight;</script>
    at various points in the html code
  • Idea by Maciej Stachowiak, published by Dave Hyatt.
  • Bugzilla has some "naive art" precedents in the testcases
  • This code clearly sets a breakpoint. .
  • "If a script stalls, it's blocking the parsing of the rest of the content, so if a layout does occur, you're basically guaranteed that you're going to lay out the content of the page exactly up to the opening script tag and no more." Dave Hyatt
