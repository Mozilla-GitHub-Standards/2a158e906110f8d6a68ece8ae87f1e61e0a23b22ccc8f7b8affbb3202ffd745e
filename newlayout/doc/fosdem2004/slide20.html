Incremental Reflow Bugs

  • Depending on the connection speed and the cache state different code paths will be executed.
  • The rendering however should be predictable and not depend on the used code path.
  • Typical reflow bugs have descriptions like: "page different on reload", which is typically caused by loading the content from the faster cache.
  • These bugs have {inc} in the bug title.
  • The bug is often not reproducible once the page has been downloaded.
  • You need to cause incremental reflows like you do when using a slow connection.
