Emphasize

  • Single pixel effects will not be easily noticeable for a layout developer, so try to adjust the values in the testcase to create a noticeable effect from the bug.
  • You have a lot of parameters that can be tweaked like,
    • the margin,
    • the padding,
    • the border,
    • the borderspacing,
    • often the content itself
  • For a far more detailed description of testcase-making techniques, read the CSS2.1 Test Case Authoring Guidelines
