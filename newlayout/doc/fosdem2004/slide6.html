A simplified testcase

A simplified test case means the simplest possible Web page that still reproduces the bug. If you remove any more characters from the file of the simplified test case, you no longer see the bug.

The Gecko bugathon

  • Simplified testcases are the lingua franca in layout
  • If you don't create a testcase people, that could otherwise fix bugs, have to spend time to create the testcase first
  • Only with a testcase can you decide that the bug report is a dupe or invalid. ( Triage of layout bugs by David Baron)
