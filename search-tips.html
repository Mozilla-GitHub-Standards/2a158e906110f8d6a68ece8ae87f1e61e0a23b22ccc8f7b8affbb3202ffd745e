Searching for Mozilla Developer Documentation

Developer documentation are scattered in mozilla.org, Mozilla source code, and various other places on Internet, thus it may be difficult to find the information you want. The following are some helpful links and search forms to get you find developer documentation quickly.

Documentation in the Source Code

The source code tree is a good place to look for up-to-date information on the source code. Sometimes you can also find documentation engineers put in the source tree.

You can search and browse the source code from the following two interfaces:

  • LXR is a massively-hyperlinked source code browser that lets you cross-reference function and variable names.
  • Deoxygen code documentation offers cross-reference class and function diagrams with descriptions compiled automatically from the source code.

Documentation not Maintained on mozilla.org Site

The following lists some useful online resources:

  • XULPlanet maintains excellent documentation on XUL and Mozilla XPCOM APIs
  • Netscape DevEdge maintains various references and guides to Mozilla technologies, such as Gecko Plug-in API Reference.
  • MozillaZine is a news and community site for Mozilla developers and users.
  • mozdev.org hosts various Mozilla-related projects, including PluginDoc and online contents for the book Creating Applications with Mozilla.

You may also search Google for Internet resources not listed here.

Postings on Mozilla Newsgroups

mozilla.org maintains various developer newsgroups which can be valuable mine of frequently requested information.
