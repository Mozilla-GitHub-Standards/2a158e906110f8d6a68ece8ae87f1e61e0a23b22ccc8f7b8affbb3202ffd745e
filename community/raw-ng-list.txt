:Mozilla Technologies

mozilla.dev.tech.crypto                 For discussions about cryptography, and cryptographic issues surrounding the Mozilla source code. See the <a href="../projects/security/pki/">PKI</a> project for more info.
mozilla.dev.tech.crypto.checkins        Join this list if you'd like to be automatically notified of all check-ins to Mozilla's crypto code.
mozilla.dev.tech.dom                    For discussions about the <a href="http://www.w3.org/DOM/">Document Object Model</a> (DOM) in Mozilla. For scripting discussions, see the <a href="../js/scripting/">scripting resources</a> page.
mozilla.dev.tech.editor                 For discussions about HTML and text editing controls, including the <a href="../editor/">Composer</a> component, <a href="../editor/midas-spec.html">rich text editing</a> and text widgets.
mozilla.dev.tech.gfx                    For people working on Mozilla's graphics subsystem, including the port to Cairo.
mozilla.dev.tech.java                   For discussions about Java: both embedding <acronym title="Java Runtime Environment">JRE</acronym>s in Mozilla and extending Mozilla itself using Java code. Also check out the <a href="#dev-tech-plugins">plug-in forum</a>.
mozilla.dev.tech.js-engine              For <a href="../js/">JavaScript engines</a>, including <a href="../js/spidermonkey/">Spidermonkey</a> (JavaScript in C) and <a href="../rhino/">Rhino</a> (JavaScript in Java). For scripting support, see the <a href="../js/scripting/">scripting resources</a>.
mozilla.dev.tech.js-engine.rhino        For discussions about <a href="../rhino/">Rhino</a>. Implementation of JavaScript written entirely in Java.
mozilla.dev.tech.layout                 For discussions about document layout in Mozilla, including <a href="../newlayout/">Gecko</a>. Also check out the <a href="#dev-tech-style">style</a> and <a href="#dev-tech-svg">SVG</a> forums.
mozilla.dev.tech.ldap                   For discussions about the C SDK, Java SDK, and PerLDAP, which are toolkits for developing <a href="../directory/">LDAP</a> applications.
mozilla.dev.tech.mathml                 For discussions about the <a href="../projects/mathml/">implementation</a> of W3C's <a href="http://www.w3.org/Math/">Mathematical Markup Language</a>.
mozilla.dev.tech.network                For discussions about the Mozilla Networking Library, a.k.a. <a href="../projects/netlib/">netlib</a>.
mozilla.dev.tech.nspr                   For discussions about the <a href="../projects/nspr/">portable runtime</a> code used by Mozilla.
mozilla.dev.tech.plugins                For discussions about <a href="../projects/plugins/">plug-ins</a>. Developers may also check out the <a href="#dev-tech-java">Java</a> and <a href="#dev-tech-nspr" title="Netscape Portable Runtime">NSPR</a> forums.
mozilla.dev.tech.rdf                    For discussions about the Resource Description Framework (<a href="http://www.w3.org/RDF/FAQ">RDF</a>) including the <a href=" http://home.netscape.com/comprod/products/communicator/future/aurora.html">Aurora</a> module.
mozilla.dev.tech.sql                    For discussions about using SQL in Mozilla.
mozilla.dev.tech.svg                    This is the project forum for implementing Scalable Vector Graphics (<a href="../projects/svg/">SVG</a>) in Mozilla.
mozilla.dev.tech.xbl                    For discussion about the XML Bindings Language (XBL).
mozilla.dev.tech.xforms                 For discussion about XForms in Firefox and Mozilla.
mozilla.dev.tech.xml                    For discussions about using XML in Mozilla.
mozilla.dev.tech.xpcom                  This forum is for discussions about the Cross-Platform Component Object Model (<a href="../projects/xpcom/">XPCOM</a>) that we are using to purvey and discover interfaces between independently versioned and loaded modules. XPCOM combined with XPIDL also provides <a  href="../scriptable/">scriptability</a>.
mozilla.dev.tech.xpinstall              For discussions about <a href="../projects/xpinstall/">installation</a> related technology in Mozilla including packaging and updating.
mozilla.dev.tech.xslt                   For discussions of use of the <a href="http://www.w3.org/TR/xslt">XSL Transformations</a> language in the Mozilla layout engine.
mozilla.dev.tech.xul                    For discussion about the XML-based User-interface Language (XUL).
