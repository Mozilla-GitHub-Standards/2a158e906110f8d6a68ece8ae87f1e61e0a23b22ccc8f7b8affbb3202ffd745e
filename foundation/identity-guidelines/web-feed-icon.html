Visual Identity Guidelines: Web Feeds Icon

Web Feeds Icon Usage

**DRAFT** Please send feedback to: rebron@mozilla.com

  • The Web Feed icon is to be displayed in its entirety.
  • The icon should not be displayed in parts, with color variations, or with other elements superimposed on top of the icon.
  • Icon size:
    • The icon should not be displayed on screen at any smaller than tbd.
    • The icon should not be displayed in print at any smaller than tbd.

Common Mistakes

The Web Feeds Icon should not be displayed in modified or displayed in ways that visually compromise the icon. The following figures highlight some common mistakes to avoid when displaying the icon.

Common Icon Mistakes (clockwise from top-left):
Incorrect icon color, non-standard orientation and sizing of icon, incorrect icon size ratio and position
