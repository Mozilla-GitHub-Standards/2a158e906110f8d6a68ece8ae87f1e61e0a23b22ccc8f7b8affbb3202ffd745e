DHTML Samples

Firefox support: please use the latest Firefox nightly builds to test the examples. Screen reader support will be announced before Autumn 2005.

IE support: DHTML accessibility requires application/xhtml+xml because it needs additional namespaces. Because IE does not support that application/xml+xhtml, the pages are served as ordinary text/html to IE. Until future versions of IE support application/xml+xhtml, it is unlikely that DHTML accessibility will function on it, unless screen reader vendors devise clever hacks to make it work. However, DHTML widgets can be made keyboard accessible for both Firefox 1.1 and IE. Still to do: the following examples need to be tested in IE and cleaned up for cross-browser usability.

  1. Checkbox built with accessible DHTML
  2. Slider built with accessible DHTML
  3. Spreadsheet and menu built with accessible DHTML
  4. Tree view built with accessible DHTML
