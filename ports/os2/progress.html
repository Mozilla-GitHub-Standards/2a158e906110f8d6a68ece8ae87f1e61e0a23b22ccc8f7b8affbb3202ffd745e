warpzilla progress

You can find the latest news as gleaned straight from the newsgroup on the regular status page. Don't forget, everyone is invited to participate: just stake your claim to a section of code and get coding!
on assignment

John Fairhurst is working on improving the XP frontend, mainly bugfixing and implementing missing features such as widget printing inside forms.

Henry Sobotka is is working on the backend and has been nominated, together with John Fairhurst, as the new warpzilla code owner.

Bill Law who was the former code owner is currently transfering ownership to Henry Sobotka due to lack of time and his involvement in the Win32 port. Also, he is preparing the branch merger of the OS/2 branch and the offical mozilla branch for the rest of the platforms. This is a really big item and we expecting it to happen in January.

Joan Touzet, maintainer of the temporary cvs server joyce is also working on WC_MOZILLA, the embeddable Warpzilla (a Presentation Manager window class).

Stefano Zambonin is currently working on implementing Asynchronous DNS on OS/2 - an important feature that is required for a really usable Warpzilla.

Read all about this and more in the newsgroup...
