NetHelp Status

NetHelp is the context-sensitive help documentation delivery system used in Netscape Communicator. NetHelp system files and content files are not included with the Mozilla source code.

NetHelp system files are freely available as part of the NetHelp software developer's kit (SDK). Developers can use these files to create their own help systems. For documentation on using NetHelp and the NetHelp SDK, go to:

http://home.netscape.com/eng/help/home/home.htm

Communicator version 4.5 will include a NetHelp-based help system. Versions after 4.5 are planned to use a new help display mechanism.
